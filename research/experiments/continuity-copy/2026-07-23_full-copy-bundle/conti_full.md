## 0. 스토리 비트 (오너 확정 2026-07-23)

1. **~8-9s (s09 립 바르기 중)**: 아주 희미한 여자 목소리, ASMR처럼 — **"somebody help me"**.
   소녀는 거의 알아채지 못한 채 계속 바른다.
2. **~17s (s12와 일치)**: 같은 목소리가 조금 더 크게 — **"help me"**. 소녀가 **놀라며** 거울을
   봤다가 → 세면대 쪽으로 시선을 옮긴다. (s12의 시선 이동 → s13 세면대 안 POV로 이어지는
   원본 편집 순서가 정확히 이 비트를 따라간 것 — 소리가 편집의 동기였다.)
3. **47.0s (s24)**: 블랙 프레임 + 전편 최대 음향 피크(RMS 14043, 1위) — **여자 비명+암전**이 하나의
   연출 비트. 편집 계획에서 s24는 생성 0콜(검은 화면 삽입).
4. **s26의 서 있는 소녀 = 쓰러진 소녀의 도플갱어** (오너 해석). s05·s28(변기 옆에 쓰러진 소녀
   탑뷰)과 연결. 제목 "girls"(복수)의 이유.
5. 잔여 오디오 피크(1.0s 도입 훅 · 26.0s 2위 · 29.0s · 39.0s · 57.5s · 62.0s)는 음악/효과로
   추정 — 시나리오에는 비트 1~4만 사건으로 반영.
