> - **자동 작성 (MR15+MR19)**: `프로듀서에게 채워달라` → **`AI Producer에게 작성 부탁`**. 동작 = 내부 엔진이 기존 맥락으로 빈 칸을 **추론→제안 카드**(기존 pendingProposal 재사용) → 승인 시 채움. **전제: 필수입력(스토리)이 최소 1회 있어야 추론 가능**, 없으면 질문으로 폴백. 불필요 설명문구 삭제. → 문구 아닌 **기능**이라 별도 작업.
> - **want/need (MR17)**: `외적 욕구(want) / 내적 욕구(need)`로 풀어씀.
> - **삭제 유도 (MR14)**: 인물/배경 삭제 시 닫기 가능한 재생성 유도 1줄.
