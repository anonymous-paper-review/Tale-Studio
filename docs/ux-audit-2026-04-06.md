# Tale Studio UX/Code Audit Report

**Date:** 2026-04-06
**Reviewer:** Angry Gourmet Mode
**Scope:** Full application (Login → Producer → Writer → Artist → Director → Editor)

---

## Critical (4)

### 1. 브라우저 뒤로가기 시 잠긴 스테이지 접근 가능
- **위치:** `studio/layout.tsx`, `handoff-button.tsx`
- **문제:** 뒤로가기/URL 직접 입력으로 잠긴 단계 진입 가능. layout에서 redirect 하지만, handoff의 DB 업데이트가 실패해도 silent fail
- **영향:** 클라이언트-서버 상태 불일치, 잘못된 프로젝트 단계에 데이터 기록

### 2. Writer 자동 생성 무한 루프 위험
- **위치:** `writer/page.tsx:55-60`
- **문제:** `autoGenTriggered` 상태가 생성 실패 시 제대로 관리 안 됨. `generateScenes()` 실패 시 재실행 가능
- **영향:** API 호출 폭주, 상태 오염

### 3. 비디오 프로바이더 전환 시 무반응 (Self-hosted 오프라인)
- **위치:** `director/page.tsx:213-215`
- **문제:** Self-hosted 서버가 오프라인일 때 버튼 클릭해도 아무 반응 없음. 에러 메시지 없음
- **영향:** 사용자 혼란

### 4. Producer → Writer 핸드오프 실패 시 피드백 없음
- **위치:** `producer/page.tsx:33-36`
- **문제:** `saveAndHandoff()` 실패해도 사용자에게 알림 없음. 버튼만 비활성화
- **영향:** 사용자가 성공한 줄 알고 넘어감

## Major (10)

### 5. 메모리 누수: blob URL 미해제
- **위치:** `artist-store.ts`, `director-store.ts`
- **문제:** `URL.createObjectURL(blob)` 호출 후 `URL.revokeObjectURL()` 미호출. 이미지 생성 반복 시 메모리 누적
- **영향:** 장시간 세션에서 브라우저 성능 저하

### 6. 동시 이미지 생성 시 상태 충돌
- **위치:** `artist-store.ts`
- **문제:** `generatingCharacterId`, `generatingLocationId` 각각 단일 값. 동시 생성 시 하나의 스피너만 표시
- **영향:** UI 혼란

### 7. Editor 드래그앤드롭 인덱스 검증 없음
- **위치:** `shot-timeline.tsx:43-50`
- **문제:** `dragIndex`, `toIndex` 범위 검증 없음. 잘못된 드래그 시 -1이나 범위 초과 인덱스 전달 가능
- **영향:** 클립 순서 오류

### 8. Shot Duration 입력 검증 부족
- **위치:** `shot-editor.tsx:99-110`
- **문제:** min/max 설정되어 있지만 붙여넣기로 9999 입력 가능. 무효 값은 5로 폴백되지만 사용자에게 알림 없음
- **영향:** 사용자 입력 무시

### 9. Debounce 중 페이지 이탈 시 데이터 손실
- **위치:** `director-store.ts:46-79`
- **문제:** 카메라/조명 변경이 500ms debounce. 변경 직후 탭 이동하면 저장 안 됨
- **영향:** 사용자 설정 손실

### 10. 프로젝트 로딩 에러 무시
- **위치:** `page.tsx:50-54`
- **문제:** `.catch(() => {})` 로 에러 완전 무시. API 실패 시 빈 목록 표시
- **영향:** 사용자가 프로젝트가 없다고 오해

### 11. 스토리 길이 검증 없음
- **위치:** `writer/page.tsx:153`
- **문제:** 글자 수만 표시, 최소/최대 길이 경고 없음. 20자로도 씬 생성 시도 가능
- **영향:** AI 생성 실패, API 비용 낭비

### 12. Character Lock 상태 미저장
- **위치:** `artist/page.tsx:146-151`
- **문제:** Lock/Unlock UI는 있지만 DB 저장 안 함. 새로고침 시 초기화
- **영향:** 데이터 손실

### 13. Dialogue durationHint 미사용
- **위치:** `shot-editor.tsx:50`
- **문제:** 입력 필드는 있지만 어디서도 사용 안 됨. 검증도 없음 (0, 음수, 1000 가능)
- **영향:** 죽은 코드, 사용자 혼란

### 14. 네트워크 실패 시 무한 대기
- **위치:** 전체 store 파일들
- **문제:** 대부분 `.catch()`에서 console.error만 하고 사용자 알림 없음. 재시도 로직 없음
- **영향:** 사용자가 실패를 인지 못하고 무한히 기다림

## Minor (16)

### 15. 접근성: 아이콘 전용 버튼에 aria-label 없음
- **위치:** sidebar.tsx, shot-timeline.tsx 등 다수
- **영향:** 스크린 리더 사용 불가

### 16. 키보드 네비게이션 포커스 표시 없음
- **위치:** `artist/page.tsx:118-133`
- **문제:** tabIndex={0}은 있지만 시각적 포커스 인디케이터 없음

### 17. 에러 메시지 가시성 부족
- **위치:** producer, writer 페이지
- **문제:** 작은 빨간 바 형태. 클릭하면 사라짐. 토스트 알림 없음

### 18. 비디오 플레이어 키보드 컨트롤 없음
- **위치:** `video-previewer.tsx`
- **문제:** 스페이스바 재생/일시정지, 화살표 탐색 미지원

### 19. 사이드바 잠긴 스테이지 구분 어려움
- **위치:** `sidebar.tsx:47`
- **문제:** opacity-30만 적용. 색상 변화 없어 구분 어려움

### 20. 프로젝트 초기화 중 로딩 인디케이터 없음
- **위치:** `studio/layout.tsx:24-35`
- **문제:** `initLoading` 중에도 children 렌더링. 데이터 깜빡임

### 21. 채팅 스크롤 타이밍 문제
- **위치:** writer-chat.tsx, director-chat.tsx
- **문제:** `setTimeout(50ms)` 매직 넘버. 빠른 메시지 시 스크롤 위치 오류

### 22. 로그인 페이지 세션 체크 없음
- **위치:** `login/page.tsx`
- **문제:** 이미 로그인된 사용자도 /login 접근 가능 (middleware에서 redirect 하지만 깜빡임)

### 23. 삭제 확인 대화상자 없음
- **위치:** `shot-timeline.tsx:129-135`
- **문제:** 클립 삭제 시 확인 없이 즉시 삭제

### 24. Color Temperature 슬라이더 그라데이션 정렬 문제
- **위치:** `key-light.tsx:102-108`
- **문제:** 수동 positioning으로 슬라이더와 그라데이션 배경 오정렬 가능

### 25. 드래그앤드롭 드롭 영역 불명확
- **위치:** `shot-timeline.tsx`
- **문제:** 드래그 오버 시 border 변화만 있어 드롭 가능 여부 불명확

### 26. Health Check 폴링 간격 너무 김
- **위치:** director/page.tsx (30초)
- **문제:** 서버 재시작 후 최대 30초간 오프라인 표시

### 27. Aspect Ratio 변경 불가
- **문제:** Producer에서 설정한 aspectRatio를 이후 단계에서 변경할 수 없음

### 28. 대규모 프로젝트 목록 페이지네이션 없음
- **위치:** `page.tsx`
- **문제:** 프로젝트 100개 이상 시 전체 로딩

### 29. Video Duration NaN 체크 없음
- **위치:** `video-previewer.tsx:45`
- **문제:** 로딩 중 duration이 Infinity/NaN일 수 있음

### 30. 빈 Scene Manifest에서 Shot 선택 시 크래시 가능
- **위치:** `director/page.tsx:91`
- **문제:** shots가 비어있는데 selectedSceneId가 설정되어 있으면 빈 리스트
