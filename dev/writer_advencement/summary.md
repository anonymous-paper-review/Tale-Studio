# Storyboard-to-Video 1:1 명세 리서치 요약

> 작성일: 2026-04-15
> 목적: AI 영상 생성 파이프라인이 "영상 1:1 대응 문서"를 만들려면 어떤 정보가 필요한지 전수 조사
> 핵심 발견: 시중 스토리보드는 전체 영상 명세의 약 30~40%만 담고 있으며, 나머지는 팀 간 공유 문서와 암묵적 지식에 의존한다.

---

## 왜 스토리보드만으로는 부족한가

제작팀은 스토리보드 이전에 캐릭터 시트, 아트 바이블, 로케이션 디자인, 컬러 스크립트 등을 **사전에 공유**한다.
따라서 스토리보드에는 "이 샷에서 뭐가 달라지는가"만 기록하고, 변하지 않는 것(캐릭터 외형, 세트, 스타일)은 생략한다.

AI 영상 생성에서는 이 "당연히 알고 있는" 모든 것을 **명시적 데이터 구조로** 보유해야 한다.

---

## 7대 생략 영역 분류

### 1. 캐릭터 명세 (Character Bible / Model Sheet)

스토리보드에 "Kai가 화난 표정"이라고만 적혀 있으면, Kai가 어떻게 생겼는지는 별도 캐릭터 시트에 존재.

| 요소 | 설명 |
|------|------|
| 턴어라운드 뷰 | 정면, 3/4, 측면, 후면 |
| 프로포션 | 머리-몸 비율, 체형, 키 |
| 표정 시트 | 기쁨/슬픔/분노/놀람/공포 등 캐릭터 고유 표현 |
| 포즈 시트 | 시그니처 포즈, 아이들, 액션 |
| 색상 팔레트 | 머리/피부/눈/의상 정확한 hex/RGB |
| 스케일 차트 | 다른 캐릭터 대비 키 비교 |
| 의상 명세 (씬별) | 어떤 옷, 어떤 액세서리, 손상 상태 |
| 성격/바디랭귀지 | 캐릭터 특유의 몸짓, 자세, 습관 |
| 음성 특성 | 톤, 악센트, 말투 속도 |

### 2. 아트 디렉션 / 스타일 바이블 (Art Direction Bible)

"이 영상이 어떤 느낌인가"에 대한 전체 규칙.

| 요소 | 설명 |
|------|------|
| 비주얼 스타일 | 실사 / 셀애니 / 3D / 페인터리 / 모션그래픽 |
| 쉐이프 언어 | 둥근형=친근, 각진형=위협적 |
| 라인 퀄리티 | 두꺼운 외곽선 / 라인리스 / 러프 / 클린 |
| 텍스처 철학 | 매끄러운 vs 거칠고 질감 있는 |
| 글로벌 컬러 팔레트 | 주조색, 보조색, 금지색 |
| 컬러 의미 체계 | 빨강=위험, 파랑=고독 등 작품 내 규칙 |
| 레퍼런스 작품 | "블레이드러너 2049 + 지브리 느낌" |
| 시대/장르 비주얼 규칙 | 느와르=하이콘트라스트+그림자, SF=쿨블루+렌즈플레어 |

### 3. 컬러 스크립트 (Color Script)

씬 전체를 관통하는 색상/조명의 감정선.

| 요소 | 설명 |
|------|------|
| 씬별 색온도 변화 | 따뜻한 -> 차가운 흐름 |
| 조명 무드 아크 | 밝음/어둠이 감정 곡선에 매핑 |
| 컬러 그레이딩 방향 | 틸앤오렌지, 디새추레이션, 빈티지 |
| 감정별 컬러 매핑 | 슬픔=뮤트 블루, 분노=채도 높은 레드 |

### 4. 로케이션/배경 명세 (Location Design Package)

스토리보드에 "사무실"이라고만 적혀 있을 때, 실제로 필요한 정보들.

| 요소 | 설명 |
|------|------|
| 건축 양식 | 시대, 문화, SF적 디자인 |
| 공간 평면도 | 벽, 문, 창문, 가구 위치 (카메라 배치 근거) |
| 재질과 텍스처 | 돌, 나무, 금속, 유리 (빛 반사에 영향) |
| 세트 드레싱 인벤토리 | 배경에 보이는 모든 물건 목록 |
| 실용적 광원 | 창문, 램프, 촛불, 네온 위치 |
| 공간 스케일 | 방 크기, 천장 높이, 복도 너비 |
| 상태 변화 | 깔끔 -> 파괴된, 낮 -> 밤 변화 |
| 날씨/대기 | 비, 눈, 안개, 먼지, 바람 방향과 세기 |

### 5. 사운드 디자인 (Sound Design Document)

스토리보드에서 거의 다루지 않는 영역.

| 요소 | 설명 |
|------|------|
| 앰비언트 사운드스케이프 | 도시 소음, 숲 새소리, 우주선 험 |
| 폴리 | 발소리, 옷 스치는 소리, 물건 조작음 |
| 음악 큐 | 장르, 템포, 악기, 감정 타겟 |
| 디제틱 vs 논디제틱 | 스토리 세계 안의 소리 vs 관객용 사운드 |
| 사운드 트랜지션 | 오디오 크로스페이드, 사운드 브릿지 |
| 의도적 정적 | 침묵이 임팩트를 만드는 순간 |

### 6. 연속성 규칙 (Continuity Rules)

경험 많은 스태프가 "당연히" 지키는 것들.

| 요소 | 설명 |
|------|------|
| 180도 룰 | 액션 축의 어느 쪽에 카메라가 있는지 |
| 스크린 디렉션 | 캐릭터 이동 방향 좌->우 일관성 |
| 아이라인 매칭 | 샷-리버스샷 시선 일치 |
| 30도 룰 | 같은 피사체 연속 샷은 최소 30도 이동 |
| 소품 연속성 | 어느 손에 컵, 잔에 물이 얼마나 |
| 의상 연속성 | 재킷 온/오프, 소매 올림/내림 |
| 감정 연속성 | 컷 사이 감정 상태 일치 |
| 시간 연속성 | 시계, 태양 위치, 그림자 길이 일관성 |

### 7. 샷 레벨 미시 디테일 (Shot-level Micro Details)

샷 단위에서 거의 항상 암묵적으로 처리되는 요소들.

| 요소 | 설명 |
|------|------|
| 초점 거리/렌즈 | 24mm=왜곡/확장, 85mm=압축/친밀, 200mm=극단 압축/관음적 |
| 피사계 심도 | 뭐가 포커스고 뭐가 흐린지 (시선 유도 핵심) |
| 랙 포커스 타이밍 | 포커스 A->B 이동 속도 |
| 모션 블러 양 | 셔터 앵글 (180도=일반, 45도=스타카토) |
| 카메라 높이 | "로우앵글"이 아니라 "지면에서 60cm" |
| 헤드룸/리드룸 | 머리 위 공간, 이동방향 앞 공간 비율 |
| 그레인/노이즈 | 클린 디지털 vs 필름 그레인, 강도 |
| 비네팅 | 프레임 가장자리 어둡게 |
| 색수차 | 가장자리 컬러 프린징 (빈티지 렌즈 효과) |
| 스킨 렌더링 | 매끄러움 vs 질감, SSS |
| 샷 내부 페이싱 | 액션 즉시 시작 vs 카메라가 잠시 머무름 |
| 표정 미세 상태 | "화남"이 아니라: 눈썹 위치, 턱 긴장, 눈 찡그림, 콧방울, 입술 |
| 바디랭귀지 | 자세, 무게 중심, 손 위치, 어깨 긴장도 |
| 샷 내 색온도 | 혼합 조명 (따뜻한 실내등 + 차가운 창문빛) |
| 샷별 컬러 그레이딩 | 채도, 콘트라스트 커브, 특정 색 시프트 |

## 영상 장르별 특이 요소

| 유형 | 고유 요소 |
|------|----------|
| 실사 영화 | 렌즈 스펙, 조리개/DOF, 필름 스톡/센서 룩, 스턴트, 분장/프로스테틱, 로케이션 퍼밋 |
| 2D 애니메이션 | 프레임레이트(12fps on 2s vs 24fps on 1s), 레이어 분리, 립싱크 방식, 이펙트 애니메이션(불/물/연기/마법) |
| 3D 애니메이션 | 리깅 요구사항, 셰이더/서페이스(SSS, 메탈릭/러프니스), 렌더링 스타일(레이트레이스/NPR/하이브리드), 시뮬레이션(천/머리카락/유체), 라이팅 리그 |
| 스톱모션 | 퍼펫 제작 스펙, 리플레이스먼트 파트(입 모양, 손), 세트 물리 치수, 모션 컨트롤 리그 |
| 모션그래픽 | 브랜드 가이드, 타이포그래피 스펙(폰트, 크기, 웨이트, 애니메이션 동작), 아이콘 스타일, 이징 커브, 그리드 시스템, CTA 배치 |
| 광고 | 법적 고지 요건, 프로덕트 샷 스펙, 엔드카드/로고 애니메이션, 플랫폼별 비율(16:9, 9:16, 1:1) |
| 뮤직비디오 | 가사<->비주얼 싱크맵, 비트/리듬 매핑, 퍼포먼스 vs 내러티브 비율, 안무 스펙 |
| 다큐멘터리 | 인터뷰 프레이밍/조명 셋업, B-roll 목록, 아카이벌 푸티지 삽입점, 재연 스타일 규정, 윤리 가이드라인 |

## 전체 보충 문서 생태계 (Production Bible)

스토리보드와 함께 있어야 완전한 영상 명세가 되는 문서 체계:

```
스토리보드 (Shot-level visual)
  +-- 스크립트/시나리오         -- 내러티브 원천
  +-- 캐릭터 모델 시트          -- 외형 일관성
  +-- 아트 디렉션 바이블        -- 비주얼 규칙
  +-- 컬러 스크립트             -- 감정 컬러 아크
  +-- 로케이션 디자인 패키지    -- 평면도, 컨셉아트
  +-- 의상/분장 바이블          -- 씬별 캐릭터 외형
  +-- 소품 목록                -- 인터랙션 오브젝트
  +-- 세트 드레싱 목록          -- 배경 오브젝트
  +-- 사운드 디자인 문서        -- 앰비언스, 폴리, 음악 큐
  +-- VFX 브레이크다운          -- 샷별 디지털 이펙트
  +-- 룩북                     -- 컬러 그레이딩 방향, LUT 레퍼런스
  +-- 무드보드                 -- 톤/느낌 레퍼런스 이미지 모음
  +-- 애니매틱/프리비즈         -- 타이밍 테스트
  +-- 샷 리스트                -- 스토리보드의 체크리스트 버전
```

## 잘 만들어진 스토리보드의 기준

1. **커버리지 완전성**: 모든 내러티브 비트에 대한 시각적 커버리지가 존재 (빈 곳 없음)
2. **연속성 무결성**: 스크린 디렉션, 아이라인, 캐릭터 스케일, 소품/의상이 패널 간 일관
3. **커뮤니케이션 명확성**: 카메라 움직임이 모호하지 않음 (화살표, 레이블)
4. **구도 품질**: Rule of thirds, 리딩라인, 깊이 스테이징(전경/중경/후경) 의도적 사용
5. **실용성**: 감독뿐 아니라 모든 부서가 읽을 수 있을 만큼 명확한 주석

## 다음 단계: 아이디에이션 방향

7대 영역 각각에 대해 디테일한 분석 / 실제 사례 조사로 확장 예정:

1. **캐릭터 명세** -- 픽사/디즈니 캐릭터 시트 사례, AI 캐릭터 일관성 문제
2. **아트 디렉션** -- 스타일 바이블 실제 구조, 레퍼런스 기반 스타일 정의 방법론
3. **컬러 스크립트** -- 픽사 컬러 스크립트 사례, 감정-색상 매핑 체계
4. **로케이션/배경** -- 세트 디자인 패키지 구조, 공간 정보의 카메라 영향
5. **사운드 디자인** -- 사운드 디자인 문서 실제 포맷, AI 오디오 생성 매핑
6. **연속성 규칙** -- 연속성 오류 사례, AI 파이프라인에서의 자동 검증 가능성
7. **샷 미시 디테일** -- 렌즈/DOF/그레이딩이 감정에 미치는 영향 분석
8. **장르별 특이 요소** -- 각 장르의 실제 프로덕션 문서 사례 비교
