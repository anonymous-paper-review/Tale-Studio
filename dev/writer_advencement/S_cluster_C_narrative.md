# Cluster C: 서사 원리 (Narrative Principles)

> 작성일: 2026-04-15
> 역할: 재료(A+B)를 "이야기"로 만드는 접착제/엔진
> 핵심: 줄거리 요약 ≠ 이야기. "X 일어남, 그래서 Y 일어남, 그것이 Z를 의미함" = 이야기

## 1. 인과 체인 (Causality Chain)

### 1.1 핵심 원칙: "그래서" vs "그리고 나서"

**사우스파크 룰 (Trey Parker & Matt Stone, 2011 NYU)**:
모든 비트 사이의 연결어를 테스트하라.

```
유효: "therefore" (논리적 귀결) 또는 "but" (합병증/반전)
무효: "and then" (시간 순서일 뿐, 인과 없음)

"and then"이 어디든 나타나면, 이야기가 아니라 이벤트 나열.
```

**예시 (브레이킹 배드)**:
월터가 암 진단받음 → **그래서** 가족 부양 위해 마약 제조 결심 → **그런데** 처남이 DEA → **그래서** 숨겨야 함 → **그런데** 파트너가 불안정 → **그래서** 상황 에스컬레이션...

**나쁜 예시**:
영웅이 괴물과 싸움 → **그리고 나서** 보물 발견 → **그리고 나서** 연인 만남 → **그리고 나서** 악당 등장 (인과 연결 없음. 이야기 아님.)

### 1.2 인과의 유형

| 유형 | 메커니즘 | 파이프라인 적용 |
|------|---------|---------------|
| **직접 인과** | A가 직접 B를 일으킴 | 샷-to-샷 (씬 내부) |
| **간접/나비효과** | A가 체인을 통해 B에 도달 | 씬-to-씬 (액트 간) |
| **캐릭터 기반** | 성격 특성이 행동을 유발 → 결과 | 캐릭터 아크 통합 |
| **주제적 인과** | 사건들이 의미로 연결 | 씬 시퀀싱의 주제 정합 |
| **환경 인과** | 세계 조건이 행동을 제약/가능케 함 | 로케이션→샷 선택 |

### 1.3 우연의 법칙 (픽사 룰 #19)

> "우연으로 캐릭터를 **곤경에 빠뜨리는** 것은 좋다.
> 우연으로 캐릭터를 **곤경에서 꺼내는** 것은 치팅이다."

| 우연 유형 | 허용 여부 | 이유 |
|----------|----------|------|
| 우연이 문제 생성 | 허용 (특히 초반) | 긴장 상승, 관객은 불운 수용 |
| 우연이 문제 해결 | 불허 (특히 클라이맥스) | 주인공의 주체성 박탈, 데우스 엑스 마키나 |
| 초반 우연 | 관대함 | 세계 아직 구축 중 |
| 후반 우연 | 매우 엄격 | 관객은 인과적 보상 기대 |

**브랜든 샌더슨 계**: "무언가로 갈등을 해결하는 작가의 능력은, 독자가 그것을 얼마나 잘 이해하고 있는지에 정비례한다."

### 1.4 인과 밀도

| 복잡도 | 스레드 수 | 사례 | 적합 길이 |
|--------|---------|------|----------|
| 단순 (동화) | 1 | 빨간 모자 | 숏폼 (30~60초) |
| 표준 (영화) | 2~4 | 대부분 헐리우드 | 미드폼 (2~5분) |
| 복잡 (시리즈) | 5~12+ | 왕좌의 게임 | 롱폼 |

### 1.5 문화별 인과 모델

| 전통 | 인과 모델 | 구조 | 해소 |
|------|---------|------|------|
| 서양 (아리스토텔레스) | 선형, 직접적 | 3막 (설정-대결-해소) | 명확한 종결 |
| 동아시아 (불교/유교) | 순환, 업보, 상호연결 | 기승전결 | 모호, 해석적 |
| 인도 (힌두) | 순환, 다르마적 의무 | 다층 + 우주적 프레임 | 의무 완수 or 업보 균형 |

**기승전결 (起承転結)**: 갈등 없는 서사 모델
- 기(起): 캐릭터/세팅 도입
- 승(承): 확립된 요소 발전
- 전(転): 모든 것을 재맥락화하는 전환 (갈등이 아니라 **대비**)
- 결(結): 전환과 기존 세계의 화해

## 2. 갈등 구조 / 에스컬레이션

### 2.1 갈등 유형

| 유형 | 대립 | 시각 표현 (V축 연결) |
|------|------|---------------------|
| 인물 vs 인물 | 직접 적대자 | OTS, 2S, 대결 프레이밍 |
| 인물 vs 자신 | 내면 투쟁 | ECU, 거울, 고립 프레이밍 |
| 인물 vs 사회 | 체제적 억압 | WS에서 개인이 구조에 왜소 |
| 인물 vs 자연 | 환경 도전 | EWS 자연 지배, 인간 왜소 |
| 인물 vs 기술 | 기계/AI | 클리니컬 조명, 기하 프레이밍 |
| 인물 vs 초자연 | 다른 세계 힘 | 로우키 조명, 왜곡 앵글 |
| 인물 vs 운명 | 우주적/실존적 | 오버헤드, 광대한 빈 공간 |

**최고의 씬 = 외적 + 감정적 + 철학적 갈등이 동시에 존재**

### 2.2 맥키의 적대 원리 (Robert McKee)

"주인공은 적대 세력이 만드는 만큼만 매력적이다."

| 코너 | 정의 | 예시 (정의 테마) |
|------|------|-----------------|
| 긍정 | 원하는 가치 | 정의 |
| 반대 (Contrary) | 다소 부정적, 완전 반대는 아님 | 불공정 (편애, 관료주의) |
| 모순 (Contradictory) | 직접 반대 | 부정의 (법 위반, 범죄) |
| 부정의 부정 | 긍정으로 위장한 부정; 질적으로 더 악화 | 폭정 ("힘이 곧 정의") |

**스토리 진행**: 긍정 → 반대(1막) → 모순(2막) → 부정의 부정(3막)
각 단계에서 갈등이 **양적이 아니라 질적으로** 심화.

### 2.3 에스컬레이션 vs 인플레이션

```
에스컬레이션 (올바름): 위험에 처한 것의 "의미"가 깊어짐
  "더 많은 사람이 위험" (X) → "사랑하는 사람이 위험" (O)

인플레이션 (잘못됨): 폭발이 커지고, 사망자가 늘고, 음악이 커짐
  → 더 큰 폭발 ≠ 더 큰 긴장
```

### 2.4 호흡 원칙

긴장은 주기적으로 해소되어야 더 강하게 재구축됨.

```
감정 강도 시간축:

    ╱╲      ╱╲╲       ╱╲╲╲
   ╱  ╲    ╱    ╲     ╱     ╲  ← 클라이맥스
  ╱    ╲  ╱      ╲   ╱       ╲
 ╱      ╲╱        ╲ ╱         ╲
╱                   ╲           ╲___

각 피크가 이전보다 높음. 각 밸리가 이전보다 얕음. 전체 추세: 상승.
```

### 2.5 거짓 승리 / 거짓 패배

| 장치 | 설명 | 효과 |
|------|------|------|
| 거짓 승리 | 잘 풀리는 듯 → 붕괴 | 이후 패배를 더 아프게 |
| 거짓 패배 | 모든 것 끝난 듯 → 길이 열림 | 이후 승리를 더 earned하게 |

패턴: "키스와 뺨 때리기" -- 좋은 이야기는 이 둘을 번갈아 배치.

### 2.6 갈등 해소 패턴

| 패턴 | 감정 효과 | 장르 친화도 |
|------|----------|-----------|
| 승리 | 승리감, 만족 | 액션, 모험 |
| 희생 | 쓸쓸함, 고귀함 | 전쟁, 영웅서사 |
| 타협 | 성숙, 뉘앙스 | 정치 드라마 |
| 변환 | 패러다임 시프트 | SF, 철학 |
| 수용 | 평화, 지혜 | 문예, 아트필름 |
| 비극 | 슬픔, 카타르시스 | 비극, 느와르 |
| 피로스 승리 | 공허, 여운 | 전쟁, 범죄 |
| 모호 | 사유, 여운 | 아트, 스릴러 |

## 3. 셋업 / 페이오프

### 3.1 체호프의 총

> "1막에서 총을 보여줬으면, 3막에서 발사해야 한다."
> 역: "3막에서 총이 발사되면, 1막에서 보여줬어야 한다."

첫 번째: 매달린 요소 방지. 두 번째: 데우스 엑스 마키나 방지.

### 3.2 셋업/페이오프 유형

| 유형 | 셋업 | 페이오프 | 사례 |
|------|------|---------|------|
| 플롯 | 오브젝트, 로케이션, 세계 규칙 제시 | 오브젝트 사용, 로케이션이 결정적 | 다이하드: 맨발 |
| 캐릭터 | 특성, 스킬, 결함 입증 | 특성 시험, 스킬 필요, 결함이 실패 유발 | |
| 주제적 | 아이디어 소개/질문 | 아이디어 도전/증명/변환 | 쇼생크: "희망은 위험" vs "희망은 좋은 것" |
| 대사 | 특정 맥락에서 말한 문구 | 같은 문구가 변형된 의미로 반복 (콜백) | 터미네이터: "I'll be back" |
| 시각 | 반복되는 이미지/모티프 | 이미지가 새 의미로 귀환 | 시민 케인: Rosebud 썰매 |

### 3.3 미묘함 스펙트럼

| 레벨 | 관객 인지 | 효과 | 리스크 |
|------|----------|------|--------|
| 텔레그래핑 | 알아차리고 예측함 | 서스펜스 소멸 | 지루함 |
| 명시적 복선 | 알아차리지만 정확히 예측 못 함 | 기대감 | 무거울 수 있음 |
| 은근한 복선 | 지나치다가 페이오프에서 "아하!" | 만족감, earned | 너무 은근하면 놓침 |
| 눈앞의 숨김 | 보이지만 의미를 인식 못 함 | 최대 "필연인데 놀라운" | 마스터리 필요 |
| 회고적 깨달음 | 재감상에서만 발견 | 재감상 보상 | 첫 감상 효과 없음 |

### 3.4 만족의 공식

| 결과 | 관객 느낌 |
|------|----------|
| **놀랍지만 돌이켜보면 필연** | 깊은 만족 -- "처음부터 계획했구나" |
| 예상대로 도착 | 편안하지만 스릴 없음 |
| 예상했지만 뒤틀림 | 기분 좋은 놀라움 |
| 페이오프 없음 | 좌절 -- "깨진 약속" |
| 셋업 없는 페이오프 | unearned, 데우스 엑스 마키나 |

## 4. 관객 경험 설계

### 4.1 히치콕: 서스펜스 vs 서프라이즈

**서프라이즈**: 관객이 모름 → 갑작스런 드러냄. 15초의 충격.
**서스펜스**: 관객이 폭탄을 봄 → 캐릭터는 모르고 야구 얘기 → 15분의 고통.

**원칙**: 가능할 때마다 관객에게 정보를 주라. 드라마틱 아이러니가 거의 항상 더 강력하다.

### 4.2 정보 비대칭

| 상태 | 관객 앎 | 캐릭터 앎 | 효과 |
|------|---------|----------|------|
| 드라마틱 아이러니 | O | X | 서스펜스, 공포, 무력감 |
| 공유 발견 | X | X | 동일시, 함께 발견 |
| 미스터리 | X | O (일부) | 호기심, 몰입 |
| 전지적 뷰 | O | O | "무엇"이 아닌 "어떻게"에 집중 |

**미스터리 박스 (JJ 에이브럼스) vs 열린 손 (히치콕)**:
- 미스터리 박스: 퍼즐 > 감정. 초기 호기심은 강하지만 답이 실망스러울 수 있음.
- 열린 손: 감정 > 퍼즐. 지속적 긴장, 감정적 깊이.
- **이야기는 퍼즐이 아니라 감정**이므로, 히치콕 접근이 일반적으로 우월.

### 4.3 "Yes, But / No, And" 매트릭스

| 응답 | 의미 | 서사 효과 | 배치 |
|------|------|----------|------|
| Yes, And | 성공 + 더 얻음 | 긍정 모멘텀 | 피날레, 작은 승리 |
| **Yes, But** | 성공 + 합병증 | 성공했지만 새 위험 | **대부분의 씬 (긴장 유지)** |
| **No, But** | 실패 + 긍정적 성장 | 좌절했지만 희망 | 캐릭터 성장 순간 |
| No, And | 실패 + 더 악화 | 복합 실패 | 가장 어두운 순간 |

**규칙**: 관객이 정확히 기대하는 것을 주지 마라. Yes, But 또는 No, But이 대부분이어야 한다.

## 5. 주제 관통

### 5.1 주제 = 메시지가 아니라 질문

```
나쁜 주제: "전쟁은 나쁘다" (메시지, 설교적)
좋은 주제: "전쟁은 평범한 사람에게 무엇을 하는가?" (질문, 탐구)
```

**균형**: 주제는 **느껴져야지 말해지면 안 됨.**

### 5.2 주제적 논증 구조

| 역할 | 기능 | 예시 (파고) |
|------|------|-----------|
| 주인공 | 미결정 위치, 어느 쪽을 선택할지 | 제리 런데가르드 |
| 적대자 | 반-주제 (지는 쪽) | 칼과 개어 |
| 스테이크스 캐릭터 | 주제의 진실 (이기는 쪽) | 마지 건더슨 |

**파고의 주제적 질문**: "돈이 행복에 필수인가?"
제리 = 어떤 대가든 돈 추구. 범죄자 = 돈을 위해 살인. 마지 = 부 없이 만족.
클라이맥스가 마지의 관점을 검증.

### 5.3 주제 표현 방법 (미묘함 순)

1. **캐릭터 행동** (가장 강력, 가장 은근): 캐릭터가 압박 속에서 **하는 것**으로 주제 표현
2. **상징적 오브젝트**: 캐릭터와 함께 진화하는 반복 시각 모티프
3. **대사** (가장 노골적, 절제 필요): 핵심 순간에 전략적으로만 배치

## 6. 관객의 감정 아크

### 6.1 보네거트의 이야기 형태 (과학적 검증)

MIT 연구자들이 1,700+ 작품 분석으로 6가지 기본 감정 아크 확인:

| 아크 | 형태 | 설명 | 사례 |
|------|------|------|------|
| 상승 | ╱ | 꾸준한 행운 증가 | 이상한 나라의 앨리스 |
| 하강 | ╲ | 꾸준한 행운 감소 | 로미오와 줄리엣 |
| 하강-상승 (구덩이 속 남자) | ╲╱ | 하락 후 회복 | 대부분의 코미디 |
| 상승-하강 (이카루스) | ╱╲ | 상승 후 추락 | 위대한 개츠비 |
| 상승-하강-상승 (신데렐라) | ╱╲╱ | 다중 피크/밸리 | **가장 사랑받는 이야기** |
| 하강-상승-하강 (오이디푸스) | ╲╱╲ | 복잡 교차 패턴 | 오이디푸스 왕 |

### 6.2 캐릭터 아크 ≠ 관객 아크

```
캐릭터 행복 + 관객 공포 = 드라마틱 아이러니 (폭탄을 모르는 캐릭터)
캐릭터 공포 + 관객 흥분 = 장르 쾌감 (영웅이 탈출할 거라 믿음)
캐릭터 무감 + 관객 비통 = 정보 비대칭 (관객만 아는 비극)
```

### 6.3 카타르시스 설계

**아리스토텔레스**: 연민과 공포의 정화.

| 유형 | 장르 친화도 |
|------|-----------|
| 안도 (위험 해소) | 스릴러, 호러, 액션 |
| 애도 (상실 인정) | 드라마, 비극 |
| 기쁨 (사랑/꿈 달성) | 로맨스, 코미디, 모험 |
| 계시 (진실 이해) | 미스터리, 철학 드라마 |
| 수용 (변경 불가한 것 포용) | 문예, 아트 |
| 의로운 분노 (부정의 폭로) | 사회 드라마, 법정 |
| 초월 (인간 한계 극복) | 에픽, 판타지, 영적 |

**핵심**: 카타르시스는 **earned될 때만** 작동. 애착 없이 죽음 씬 → 조작적. 투자 후 죽음 씬 → 정화.

## 7. 핍진성 규칙 종합

### 7.1 핍진성 ≠ 리얼리즘

핍진성은 **이야기 자체 규칙 내에서의 진실 외관**. 드래곤이 있는 판타지도 내적 일관성이 있으면 핍진성 완벽. 사실주의 드라마도 캐릭터가 확립된 성격에 반하면 핍진성 붕괴.

### 7.2 12가지 핍진성 위반 유형 (플롯 홀)

| # | 유형 | 위반 원칙 |
|---|------|----------|
| 1 | 논리적 비일관성 | 내적 일관성 |
| 2 | 캐릭터 동기 부재 | 캐릭터 기반 인과 |
| 3 | 연속성 오류 | 셋업/페이오프 무결성 |
| 4 | 시간 역설 | 인과 논리 |
| 5 | 설명 없는 맥거핀 | 셋업 요건 |
| 6 | 스킬셋 초과 | 캐릭터 셋업 |
| 7 | 기술 부정확 | 세계 규칙 일관성 |
| 8 | 정보 누락 | 정보 관리 |
| 9 | 편의적 해결 | earned 해소 / 인과 |
| 10 | 플롯용 장애물 | 유기적 갈등 |
| 11 | 규칙 위반 | 내적 일관성 |
| 12 | 우연 남용 | 인과 / 주인공 주체성 |

### 7.3 절대 규칙 (위반하면 항상 "거짓"으로 느껴짐)

1. **인과 규칙**: 모든 주요 사건에 추적 가능한 원인. "그래서"지 "그리고 나서"가 아님.
2. **일관성 규칙**: 확립된 규칙은 설명 없이 깨지면 안 됨. 캐릭터는 성격에 일관.
3. **주체성 규칙**: 주인공이 결과에 선택으로 영향. 외부 해결은 의미를 빼앗음.
4. **비례 규칙**: 감정 페이오프는 감정 셋업에 비례. earn되지 않은 카타르시스 불가.
5. **약속 규칙**: 모든 셋업은 페이오프. 모든 페이오프는 셋업. 깨지면 깨진 계약.
6. **에스컬레이션 규칙**: 위험은 (추세적으로) 증가. 이유 없는 후퇴 = 안티클라이맥스.

### 7.4 숙련되면 깰 수 있는 규칙

1. 우연 규칙 (코미디 의도적 부조리, 인사이팅 인시던트)
2. 갈등 규칙 (기승전결은 대비 기반, 갈등 불필요)
3. 해소 규칙 (모호한 엔딩이 주제에 봉사할 때)
4. 선형성 규칙 (비선형이 의미를 만들 때)
5. 장르 계약 (장르 전복이 메타적 가치를 가질 때)

### 7.5 6원리 상호작용

```
                주제 (5)
               ╱        ╲
              ╱          ╲
     인과 (1) ──────── 갈등 (2)
          │    ╲        ╱    │
          │     ╲      ╱     │
          │   셋업/페이오프   │
          │       (3)        │
          │     ╱      ╲     │
          │    ╱        ╲    │
 관객설계 (4) ──────── 감정아크 (6)

- 인과+갈등: 갈등이 엔진, 인과가 엔진을 바퀴에 연결
- 셋업/페이오프+인과: 페이오프 = 셋업의 인과적 해소 (같은 원리, 다른 시간 스케일)
- 주제+전부: 주제가 모든 것을 통합. 인과→주제적 논증, 갈등→주제 시험, 셋업→주제 씨앗
- 관객설계+감정아크: 나머지 4원리가 잘 되면 이 둘은 자연스럽게 발생
```

### 7.6 장르별 핍진성 관용도

| 장르 | 규칙 위반 관용도 | 핵심 핍진성 요구 |
|------|----------------|----------------|
| 리얼리즘/드라마 | 매우 낮음 | 캐릭터 심리가 진짜처럼 |
| 스릴러/미스터리 | 낮음 | 플롯 논리 완벽, 단서 공정 |
| 판타지/SF | 중간 (세계규칙), 낮음 (캐릭터) | 내적 세계 일관성 |
| 코미디 | 중~높음 | 감정 진실 > 논리 진실 |
| 호러 | 중간 | 분위기와 캐릭터 반응이 진짜 |
| 초현실/아트 | 높음 | 주제적/감정적 일관성이 논리 대체 |
