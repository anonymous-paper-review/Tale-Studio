# L1 + L2 딥다이브: 초고비용 → 고비용(부분 수정 가능)

> 작성일: 2026-04-15
> L1(변경=리메이크) 결정이 L2(변경=해당 영역 재작업)를 어떻게 제약하는가

---

## L2 요소 정의

L2는 "변경하면 관련된 씬/캐릭터/로케이션만 재작업" -- 전체 리메이크는 아니지만 비용이 큼.

| ID | 요소 | 변경 시 영향 범위 |
|----|------|------------------|
| L2-CP | 글로벌 컬러 팔레트 | 컬러 스크립트 전체 재조정, 그레이딩 재작업 |
| L2-CM | 컬러 의미 체계 | 스토리 전반 컬러 내러티브 재설계 |
| L2-LO | 로케이션 디자인 | 해당 로케이션 등장 씬 전부 |
| L2-WD | 캐릭터 의상/분장 | 해당 캐릭터 등장 샷 전부 |
| L2-SD | 사운드 디자인 기조 | 전체 오디오 재작업 (비주얼과 독립이라 비용 분리) |
| L2-VX | VFX 스타일/접근 | VFX 포함 샷 전부 |
| L2-TY | 타이포그래피/텍스트 | 텍스트 등장 샷 전부 |
| L2-PD | 소품 디자인 언어 | 해당 소품 등장 샷 전부 |

## PART 1: L2 요소별 상세 분석

### L2-CP: 글로벌 컬러 팔레트

#### 하위 결정

| 요소 | 설명 |
|------|------|
| **주조색 (Dominant)** | 영상 전체를 지배하는 1~2색 |
| **보조색 (Secondary)** | 주조색을 지원하는 2~3색 |
| **액센트색 (Accent)** | 시선 유도/포인트용 소량 사용 색 |
| **금지색 (Forbidden)** | 작품 내 절대 사용 안 하는 색 |
| **채도 범위** | 전체적으로 비비드 vs 뮤트 vs 파스텔 |
| **명도 범위** | 하이키(밝은) vs 로우키(어두운) vs 풀레인지 |
| **색온도 기준선** | 전반적으로 따뜻한 vs 차가운 |
| **대비 전략** | 보색 대비, 유사색 조화, 삼각 배색 등 |
| **팔레트 변주 규칙** | 씬/감정에 따라 어디까지 벗어날 수 있는지 |

#### L1이 L2-CP를 제약하는 방식

```
L1[S] 아트 스타일 ══> L2-CP 컬러 팔레트
  - 포토리얼 → 자연스러운 색 범위 내에서만 (극단 채도 부자연)
  - 셀셰이드 → 플랫 컬러, 그래디언트 제한
  - 페인터리 → 색 혼합/블리드 허용, 경계 부드러움
  - 애니메 → 하이 채도 + 깨끗한 분리 허용

L1[T] 텍스처 철학 ══> L2-CP
  - 프리스틴 → 깨끗한 색상 유지
  - 그런지 → 채도 떨어지고 더러움 섞임
  - 페인터리 텍스처 → 색 경계 흐려짐
```

#### 사례 연구

**인사이드 아웃 (픽사)**: 감정 캐릭터 = 색상 자체가 아이덴티티
- Joy = 노랑/파랑 글로우, Sadness = 파랑, Anger = 빨강, Disgust = 초록, Fear = 보라
- 기억 구슬 색상이 감정과 1:1 매핑
- Riley의 세계: 따뜻한 팔레트(행복한 기억) vs 점점 회색화(우울)

**웨스 앤더슨**: 영화마다 극도로 제한된 팔레트
- 그랜드 부다페스트: 핑크/보라/파랑/크림
- 문라이즈 킹덤: 카키/옐로/레드/올리브
- 모든 소품, 의상, 세트가 팔레트 내에서 통제됨

**브레이킹 배드**: 의도적 컬러 심볼리즘
- 월터 화이트: 연한 색(순수) → 점점 어두운 색(타락)
- 스카이러: 파랑(정직/우울)
- 제시: 빨강/노랑(위험/에너지)
- 행크: 오렌지/갈색(법/질서)
- 각 캐릭터의 컬러가 씬마다 변화하며 아크를 시각적으로 전달

**매드 맥스 퓨리 로드**: 극단적 2톤 접근
- 낮: 오렌지/옐로 (사막, 불, 전쟁)
- 밤: 틸/블루 (쿨, 안식, 도피)
- 이 2색 대비가 영화 전체를 관통
- 컬러리스트의 "그래픽 노블" 팔레트 철학: 디새추레이트 포스트아포 대신 리치 채도

**Her (2013)**: 금지색의 교과서
- 프로덕션 디자이너 K.K. Barrett 규칙: "모든 프레임에 약간의 빨강"
- **파랑 완전 금지** -- 디스토피아 느낌 방지
- 결과: 레드/핑크/옐로의 따뜻한 파스텔 세계 = 부드러움과 노스탈지아
- 쿨톤은 오직 고립 순간에만 허용

#### 숨겨진 하위 결정 (에이전트 리서치 보강)

| 결정 | 설명 |
|------|------|
| **"화이트" 정의** | 이 작품에서 흰색은 따뜻한 흰? 블루 흰? 크림? |
| **그림자 색** | 그림자는 순수한 검정이 아님 -- 보라/파랑/갈색 캐스트 |
| **앰비언트 색** | 조명 적용 전 공간의 기본 채움 색 |
| **컬러 "무게"** | 팔레트 내 각 색의 시각적 중력 (빨강이 지배? 액센트?) |
| **씬 간 컬러 전환 규칙** | 점진적 블렌드? 하드컷? |

### L2-CM: 컬러 의미 체계

#### 하위 결정

| 요소 | 설명 |
|------|------|
| **캐릭터별 컬러 코드** | 각 캐릭터에 배정된 시그니처 색상 |
| **진영/세력 컬러** | 선/악, 아군/적군 색상 분리 |
| **감정별 컬러** | 기쁨=따뜻한, 슬픔=차가운, 분노=빨간 등 |
| **시간/장소 컬러** | 과거=세피아, 현재=풀컬러, 미래=블루틴트 |
| **컬러 아크** | 캐릭터/스토리 진행에 따른 색상 변화 궤적 |
| **전환 시그널** | 컬러 변화가 내러티브 전환을 알림 |

#### L1이 L2-CM을 제약하는 방식

```
L1[S] 아트 스타일 ══> L2-CM
  - 포토리얼 → 컬러 의미가 미묘해야 함 (빨간 넥타이 수준)
  - 카툰/애니메 → 노골적 컬러 코딩 가능 (캐릭터=색상)
  - 추상 → 컬러가 형태를 대체할 수 있음
```

#### 사례 연구

**아케인 (Piltover vs Zaun)**: 도시 자체가 컬러로 분리
- Piltover (위): 골든/앰버/따뜻한 자연광, 깨끗, 질서
- Zaun (아래): 네온 그린/퍼플/독성 발광, 더럽고, 혼돈
- Jinx의 컬러: Zaun 네온 핑크/블루 → 정신 불안정 심화와 함께 채도 극대화
- Vi의 컬러: 핑크 → 어둠/빨강(분노)으로 전이

**마블 MCU**: 캐릭터 컬러 코딩의 교과서
- 아이언맨=빨강/골드, 캡틴 아메리카=파랑/빨강/흰, 헐크=초록
- 타노스=보라 (왕권/극단), 인피니티 스톤 6색이 각각 고유 의미

**대부 (The Godfather)**: 오렌지 = 죽음
- 살인이 일어나기 전 씬마다 오렌지가 등장 (과일 바구니, 포장지)
- 관객이 의식하지 못하지만 반복으로 불안감 학습

**판의 미로 (Pan's Labyrinth)**: 세계관별 온도 분리
- 판타지 세계: 따뜻한 색 (크림슨, 골든 앰버) = 상상력/탈출
- 현실: 차가운 색 (블루, 그레이) = 억압/죽음

**브레이킹 배드 컬러 믹싱**: Skyler의 블루 + Jesse의 옐로 = Walt의 그린
- 이 색 혼합 관계가 캐릭터 관계를 은유

#### 숨겨진 하위 결정

| 결정 | 설명 |
|------|------|
| **컬러 "문법"** | 반복으로 관객에게 가르치는 색 읽기 규칙 (명시 안 됨) |
| **채도 = 의미** | 디새추레이션 = 감정 위축, 하이퍼채도 = 감정 절정 |
| **컬러 오염 타이밍** | 한 캐릭터의 색이 다른 캐릭터 공간에 등장하는 시점 (동맹/타락 시그널) |
| **리셋 컬러** | 감정 피크 후 팔레트가 돌아가는 뉴트럴 색 |

### L2-LO: 로케이션/환경 디자인

#### 하위 결정

| 요소 | 설명 |
|------|------|
| **건축 양식** | 시대, 문화, 판타지 디자인 철학 |
| **공간 평면도** | 벽, 문, 창, 가구 배치 → 카메라 가능 위치 결정 |
| **재질/표면** | 돌, 나무, 금속, 유리 → 빛 반사/흡수 결정 |
| **실용적 광원** | 창문, 램프, 네온 → 조명의 물리적 근거 |
| **세트 드레싱 인벤토리** | 배경에 보이는 모든 물건 |
| **공간 스케일** | 방 크기, 천장 높이 → 캐릭터 대비 느낌 |
| **컬러 스킴 (로케이션별)** | 글로벌 팔레트 내에서의 로케이션 고유 색상 |
| **상태 변화 스펙** | 시간대, 날씨, 파괴/변형 단계 |
| **소리 특성** | 잔향(교회 vs 야외), 앰비언스 |

#### L1이 L2-LO를 제약하는 방식

```
L1[H] 쉐이프 언어 ══> L2-LO
  - 원형 세계관 → 둥근 건축, 아치, 곡선 지형 (샤이어)
  - 각진 세계관 → 직선 건축, 뾰족한 첨탑 (고담)
  - 프로포션 시스템 → 환경 스케일 (7:1 캐릭터 = 문 높이 비례)

L1[T] 텍스처 철학 ══> L2-LO
  - 프리스틴 → 깨끗한 벽, 광택 바닥
  - 리브드인 → 얼룩, 긁힘, 세월 흔적
  - 그런지 → 녹, 이끼, 금, 먼지
```

#### 사례 연구

**지브리 환경 디자인**: "살아있는 배경"
- 배경이 캐릭터보다 더 디테일한 전통 (카오나시의 목욕탕, 하울의 움직이는 성)
- 자연과 인공의 공존이 핵심 디자인 원칙
- 식물, 물, 빛의 표현에 극도의 관찰력

**블레이드 러너 2049**: 로케이션 = 감정 상태
- LA 도시: 오렌지/갈색 스모그 (디스토피아, 혼돈)
- 워커의 아파트: 차가운 블루/그레이 (고독)
- 라스베가스 폐허: 오렌지/앰버 (과거의 잔해)
- 바다벽: 회색/차가운 (종말)

### L2-WD: 캐릭터 의상/분장

#### 하위 결정

| 요소 | 설명 |
|------|------|
| **의상 디자인** | 가먼트, 소재, 패턴, 핏 |
| **컬러 (캐릭터 팔레트 내)** | L2-CM에서 배정된 캐릭터 색상 적용 |
| **씬별 변주** | 스크립트 데이마다 의상 상태 변화 |
| **손상/변형 단계** | 깨끗 → 찢어짐 → 피묻음 등 |
| **액세서리** | 모자, 안경, 시계, 무기 |
| **헤어/분장** | 헤어스타일, 메이크업, 상처, 노화 |
| **변신/각성 의상** | 캐릭터 아크에 따른 의상 전환 |

#### L1이 L2-WD를 제약하는 방식

```
L1[C] 캐릭터 코어 ══> L2-WD
  - 프로포션 → 의상 핏 (2:1 치비 = 단순 의상 / 8:1 리얼 = 복잡 의상 가능)
  - 실루엣 → 의상이 실루엣을 유지/강화해야 (캡틴의 어깨 강조)

L1[H] 쉐이프 언어 ══> L2-WD
  - 원형 캐릭터 → 부드러운 옷, 둥근 버튼, 흐르는 천
  - 각진 캐릭터 → 갑옷, 날카로운 칼라, 구조적 의상

L1[L] 라인 퀄리티 ══> L2-WD
  - 두꺼운 외곽선 스타일 → 의상 디테일 단순화 필수
  - 라인리스 → 의상 가장자리가 색상/명도로만 분리
```

#### 사례 연구

**엘사 (겨울왕국)**: 의상 = 캐릭터 아크
- 대관식 드레스 (억압) → 아이스 드레스 (해방) → 겨울왕국2 흰 드레스 (완성)
- 각 의상이 캐릭터의 내면 상태를 외면으로 표현

### L2-SD: 사운드 디자인 기조

#### 하위 결정

| 요소 | 설명 |
|------|------|
| **리얼리즘 정도** | 다큐멘터리급 리얼 vs 하이퍼스타일라이즈드 |
| **앰비언스 접근** | 밀도 (미니멀 vs 맥시멀), 레이어 수 |
| **폴리 철학** | 리얼 녹음 vs 합성/과장 |
| **음악 장르/스타일** | 오케스트라, 신스, 미니멀, 록 |
| **다이내믹 레인지** | 조용한 순간과 큰 순간의 차이 |
| **디제틱 비율** | 세계 내 소리 vs 관객용 사운드 비중 |
| **정적 사용** | 의도적 침묵의 빈도와 효과 |
| **사운드 모티프** | 캐릭터/상황별 반복 소리 |

#### L1이 L2-SD를 제약하는 방식

```
L1[S] 아트 스타일 ══> L2-SD (간접적이지만 강력)
  - 포토리얼 → 리얼리스틱 사운드 기대
  - 카툰 → 과장된/코믹 사운드 허용
  - 추상 → 실험적/앰비언트 사운드
  - 애니메 → 관례적 SE (땀방울 '픽', 놀람 '갸!' 등)
```

### L2-VX: VFX 스타일/접근

#### 하위 결정

| 요소 | 설명 |
|------|------|
| **파티클 시스템** | 불, 연기, 먼지, 마법, 빛 -- 스타일 |
| **디스토션/글리치** | 화면 왜곡 효과의 빈도와 스타일 |
| **합성 방식** | 실사 합성 vs 풀 CG vs 핸드드로운 |

```
  │                                                             │
  ├──> L2-CM 컬러 의미: 팔레트 안에서만 의미 부여 가능             │
  ├──> L2-LO 로케이션: 로케이션별 컬러 스킴이 팔레트 내            │
  ├──> L2-WD 의상: 캐릭터 의상 색이 팔레트 내                     │
  ├──> L2-VX VFX: 이펙트 색상이 팔레트 내                         │
  └──> L2-TY 타이포: 텍스트 색상이 팔레트 내                       │
                                                                 │
L2-CM 컬러 의미 ──────────> L2-WD 의상 (캐릭터 색 배정)           │
                  ──────> L2-LO 로케이션 (장소 색 배정)            │
                                                                 │
L2-LO 로케이션 ──────────> L2-SD 사운드 (잔향, 앰비언스)          │
              ──────────> L2-PD 소품 (시대/문화 일치)              │
                                                                 │
L2-WD 의상 ──────────────> L2-PD 소품 (캐릭터 소품 디자인 일치)   │
                                                                 │
L2-SD 사운드 ────────────────────── (비주얼과 독립, 교차 약함) ────┘
```

## PART 3: L2 변경의 실제 비용

| L2 요소 | 변경 시 영향받는 것 | 영향 안 받는 것 | 비용 등급 |
|---------|-------------------|----------------|----------|
| 글로벌 팔레트 | 컬러 스크립트, 모든 그레이딩, 의상 색 | 쉐이프, 프로포션, 라인 | 높음 (그레이딩으로 일부 커버) |
| 컬러 의미 | 스토리 전반 컬러 내러티브 | 물리적 에셋 | 중간 (리그레이딩 가능) |
| 로케이션 | 해당 로케이션 등장 씬만 | 다른 로케이션 씬 | 중간~높음 |
| 의상 | 해당 캐릭터 등장 샷 | 다른 캐릭터, 배경 | 중간 |
| 사운드 기조 | 전체 오디오 | 비주얼 전체 | 높지만 비주얼과 독립 |
| VFX 스타일 | VFX 포함 샷 | VFX 없는 샷 | 중간 |
| 타이포 | 텍스트 등장 씬 | 텍스트 없는 씬 | 낮음 |
| 소품 | 해당 소품 등장 샷 | 소품 없는 샷 | 낮음 |

## PART 4: 컬러 스크립트 (L1과 L2의 교차점)

컬러 스크립트는 L1(아트 스타일)과 L2(컬러 팔레트/의미)가 만나는 핵심 문서.

```
씬 1        씬 2        씬 3        씬 4        씬 5        씬 6
[따뜻한]    [밝은]      [차가운]    [어두운]    [극어둠]    [따뜻한]
[노랑]      [하늘]      [파랑]      [남색]      [검정]      [오렌지]
  ↑           ↑           ↑           ↑           ↑           ↑
 희망        발견        의심        위기        절정        회복

감정 곡선:  ───╲       ╱───╲               ╱───
                ╲     ╱     ╲     ╱       ╱
                 ╲   ╱       ╲   ╱       ╱
                  ╲ ╱         ╲ ╱       ╱
                   V           V───────╱
```

**픽사의 컬러 스크립트 프로세스**:
1. 아트 디렉터가 전체 스토리를 색상으로 시각화 (가로로 긴 하나의 그림)
2. 각 씬의 주조 색상/명도/채도/색온도를 결정
3. 감정 곡선과 색상 곡선을 매칭
4. 이 문서가 모든 부서의 컬러 레퍼런스가 됨

**코코 (픽사)**: 이중 세계 컬러 디자인
- 현실 세계: 따뜻한 멕시코 색감 (오렌지/레드/골드/테라코타)
- 죽은 자의 땅: 네온+글로우 (자주/마젠타/시안/골드 -- 870만 조명으로 표현)
- Miguel이 죽은 자의 땅에 진입하는 순간의 컬러 전환이 핵심

**소울 (픽사)**: 세계관별 완전히 다른 팔레트
- NYC 재즈 세계: 따뜻한 앰버/오렌지/브라운 (생생함, 열정)
- 그레이트 비포: 파스텔 민트/라벤더/에테리얼 블루 (영혼, 순수)
- 컬러 전환이 감정 전환을 1:1로 전달

## PART 5: 암묵적 L1-L2 경계 결정

프로가 "당연히" 내리지만 문서에 명시 안 되는 L1↔L2 경계 결정들.

| 암묵 결정 | 설명 | 영향 |
|-----------|------|------|
| **계절/시간대 컬러 규칙** | 봄=연초록, 가을=오렌지 같은 기본 매핑 | L2-CP, L2-LO |
| **컬러 팔레트 히에라키** | 어떤 색이 다른 색보다 "강한지" | L2-CP, L2-CM |
| **의상 내러티브 변환 포인트** | 의상이 바뀌는 시점 = 캐릭터 전환점 | L2-WD |
| **소품 포그라운드 규칙** | 중요 소품은 배경과 색 대비로 부각 | L2-PD, L2-CP |
| **VFX 과장 정도** | 같은 폭발이라도 리얼 vs 카툰 세기 차이 | L2-VX, L1[S] |
| **사운드-비주얼 싱크 밀도** | 모든 움직임에 소리? 핵심만? | L2-SD |
| **환경 사운드 리얼리즘** | 도시 소음 레벨이 대사에 영향 | L2-SD, L2-LO |
| **텍스트-세계관 일치** | 다이어제틱 텍스트(간판)의 서체/언어 | L2-TY, L2-LO |
