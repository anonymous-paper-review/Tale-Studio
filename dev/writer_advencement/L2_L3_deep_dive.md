## L3 요소 정의

L3은 "샷 단위 또는 씬 단위로 독립 조절 가능" -- 변경 비용이 해당 샷에 한정됨.

| ID | 요소 | 변경 시 영향 범위 |
|----|------|------------------|
| L3-CA | 카메라 워크 | 해당 샷만 |
| L3-LT | 조명 셋업 | 해당 씬/샷만 |
| L3-CG | 샷별 컬러 그레이딩 | 해당 샷만 |
| L3-PF | 캐릭터 퍼포먼스 | 해당 샷만 |
| L3-DL | 대사 딜리버리 | 해당 샷만 |
| L3-SX | 사운드 이펙트 (per-shot) | 해당 샷만 |
| L3-MU | 음악 큐 | 해당 씬/시퀀스 |
| L3-DR | 세트 드레싱 변주 | 해당 샷만 |

| 파라미터 | 설명 |

| **속도** | 느린(명상적) / 보통 / 빠른(긴박) |

| **시작-끝 위치** | 정확한 3D 좌표 |
| **동기 (Motivation)** | 왜 카메라가 움직이는가 (캐릭터 따라감 / 정보 드러냄 / 감정 전달) |
| **카메라 높이** | 정확한 cm 단위 |
| **렌즈/초점거리** | 24mm(왜곡/확장) ~ 200mm(압축/관음) |

#### L2가 L3-CA를 제약하는 방식

```
L2-LO 로케이션 ══> L3-CA
  - 공간 평면도 → 카메라 배치 가능 위치 물리적 제한
  - 천장 높이 → 크레인/하이앵글 가능 여부
  - 창문/문 위치 → 자연광 방향 고정 → 카메라 위치 영향

L2-WD 의상 ══> L3-CA
  - 화려한 의상 → CU보다 FS/MFS로 보여주고 싶은 동기
  - 마스크/헬멧 → 표정 CU 불가 → 바디랭귀지 중심 프레이밍
```

#### 사례 연구

**로저 디킨스 조명+카메라 철학**: "빛이 이미 존재하는 것처럼"
- 1917: 원테이크(처럼 보이는) 전체. 카메라가 캐릭터와 함께 이동하며 관객을 전장에 배치. 스테디캠+크레인+핸드오프의 복합.
- 블레이드 러너 2049: 극도의 네거티브 스페이스. 인물이 프레임의 작은 부분만 차지. 와이드 렌즈 + 깊은 포커스.
- 스카이폴: 실루엣 조명. 상하이 빌딩 씬에서 인물이 네온 반사 속 검은 실루엣.

**스필버그의 블로킹 & "스필버그 페이스"**:
- 캐릭터를 프레임 내에서 이동시켜 컷 없이 정보 전달 (원 테이크 블로킹)
- "스필버그 페이스": 캐릭터가 경이로운 것을 바라볼 때 CU → 리액션 → 대상 reveal
- 카메라가 캐릭터 뒤에서 시작 → 캐릭터가 발견 → 카메라가 캐릭터 앞으로 이동

**미야자키의 "마(間)"**:
- 의도적 빈 시간. 아무 일도 안 일어나는 샷.
- 캐릭터가 가만히 앉아 바람을 느끼는 3초짜리 샷
- 내러티브가 아닌 체험을 위한 카메라 — 스태틱, 와이드, 긴 홀드

### L3-LT: 조명 셋업

#### 하위 결정

| 요소 | 설명 |
|------|------|
| **키 라이트** | 주광원 -- 위치, 강도, 색온도, 하드/소프트 |
| **필 라이트** | 보조광 -- 키:필 비율 (1:1=플랫, 8:1=드라마틱) |
| **림/백 라이트** | 윤곽광 -- 피사체 분리 |
| **프랙티컬 라이트** | 프레임 내 보이는 광원 (램프, 창, 화면) |
| **앰비언트** | 전체적 채움 (자연광, 반사광) |
| **컬러 젤** | 광원별 색온도 의도적 변경 |
| **그림자 특성** | 하드 에지 vs 소프트 에지, 방향 |
| **빛의 동기** | 어디서 오는 빛인지 물리적 정당화 |

**키:필 비율과 감정:**

| 비율 | 느낌 | 장르 |
|------|------|------|
| 1:1 | 플랫, 코미디, 밝은 | 시트콤, 뮤직비디오 |
| 2:1 | 부드러운 드라마 | 로맨스, 드라마 |
| 4:1 | 드라마틱 | 스릴러, 액션 |
| 8:1+ | 극단적 명암 | 느와르, 호러 |

#### L2가 L3-LT를 제약하는 방식

```
L2-LO 로케이션 ══> L3-LT
  - 실용적 광원 위치 → 키라이트 방향의 물리적 근거
  - 창문 크기/방향 → 자연광 양과 방향
  - 재질 → 반사/흡수 (유리벽 = 반사 많음, 나무 = 흡수)
  - 공간 크기 → 빛의 확산 범위

L2-CP 컬러 팔레트 ══> L3-LT
  - 따뜻한 팔레트 → 따뜻한 조명 (3200K 텅스텐 쪽)
  - 차가운 팔레트 → 차가운 조명 (5600K+ 데이라이트 쪽)

L2-WD 의상 ══> L3-LT
  - 어두운 의상 → 림라이트로 분리 필요
  - 반짝이는 의상 → 스페큘러 하이라이트 통제 필요
```

#### 사례 연구

**루베즈키 (이매뉴얼 루베즈키) 자연광**:
- 레버넌트: 오직 자연광 + 불빛만 사용. 매직아워/골든아워에만 촬영.
- 그래비티: 우주의 유일한 광원 = 태양. 극단적 명암비.
- 버드맨: 자연광 + 실용적 광원만. 연극 무대 조명의 리얼리즘.

**핀처의 "탑라이트"**:
- 캐릭터 바로 위에서 조명 → 눈 아래 그림자 (소켓 섀도)
- 불안, 위협, 비밀의 느낌
- Se7en, Zodiac, Gone Girl에서 일관되게 사용

### L3-CG: 샷별 컬러 그레이딩

#### 하위 결정

| 요소 | 설명 |
|------|------|
| **리프트/감마/게인** | 어둡/중간/밝은 영역 색 시프트 |
| **채도 조절** | 전체 또는 특정 색 채도 |
| **콘트라스트 커브** | S-커브 강도 |
| **색온도 시프트** | 전체적 따뜻함/차가움 |
| **특정 컬러 HSL** | 개별 색상 조정 (피부톤 보호 등) |
| **블랙 레벨** | 검정이 얼마나 짙은지 (크러쉬드 vs 리프트드) |
| **하이라이트 처리** | 클립 vs 롤오프 vs 블룸 |
| **비네팅** | 가장자리 어둡게 |
| **그레인 추가** | 필름 그레인 강도 |
| **LUT/룩** | 사전 정의 색 변환 테이블 |

#### L2가 L3-CG를 제약하는 방식

```
L2-CP 컬러 팔레트 ══> L3-CG
  - 글로벌 팔레트 = 그레이딩의 "울타리"
  - 팔레트 외 색상으로 그레이딩 → 프로젝트 일관성 깨짐

L2-CM 컬러 의미 ══> L3-CG
  - "슬픔=블루" 규칙 → 슬픈 씬은 블루 쪽으로 시프트
  - 캐릭터 컬러 코드 → 해당 캐릭터 피부톤/의상톤 보호
```

#### 사례 연구

**왕가위 + 크리스토퍼 도일**:
- 화양연화: 극도의 따뜻한 앰버/레드 그레이딩 + 스텝프린팅(프레임 반복)
- 청킹 익스프레스: 네온 블루/그린 + 하이 콘트라스트
- 모든 샷이 "감정 색안경"으로 그레이딩됨

### L3-TR: 트랜지션/편집

#### 하위 결정

| 트랜지션 | 느낌 | 용도 |
|----------|------|------|
| 하드컷 | 즉시, 충격, 기본 | 대부분의 컷 |
| 점프컷 | 시간 생략, 불안 | 같은 앵글 내 시간 스킵 |
| 매치컷 | 연결, 유사성 | 시각적 유사 요소로 씬 전환 |
| J-컷 | 다음 씬 소리 먼저 | 부드러운 전환 |
| L-컷 | 이전 씬 소리 남음 | 여운 |
| 디졸브 | 시간 경과, 몽환 | 느린 전환 |
| 페이드 투 블랙 | 종결, 구분 | 챕터/액트 전환 |
| 와이프 | 장면 전환 (레트로) | 스타워즈, 코미디 |
| 스매시컷 | 극단적 대비 | 코미디, 충격 |
| 아이리스 | 집중/시대감 | 무성영화 레퍼런스 |

**편집 리듬/페이싱:**

| 패턴 | 효과 |
|------|------|
| 짧은-짧은-짧은-긴 | 긴장 축적 → 해소 |
| 점점 짧아지는 컷 | 가속, 클라이맥스 접근 |
| 점점 길어지는 컷 | 감속, 이완, 반성 |
| 일정한 리듬 | 행진, 반복, 일상 |
| 불규칙 리듬 | 혼란, 불안 |

#### 사례 연구

**에드거 라이트의 편집**:
- 숀 오브 더 데드, 핫 퍼즈, 스콧 필그림: 극단적 빠른 컷 + 매치컷 + 사운드 브릿지
- 일상 동작(아침 루틴)을 극도로 빠른 인서트로 압축
- 비트에 맞춘 컷 타이밍 (뮤직비디오적 편집)

**미야자키의 "마(間)"**:
- "아무것도 안 일어나는" 순간을 의도적으로 편집에 넣음
- 토토로: 버스 정류장에서 기다리는 장면 -- 30초간 비 떨어지는 소리만
- 이 "빈 시간"이 세계의 공기를 관객에게 전달

## PART 2: L3 내부 교차 의존성

```
L3 요소들 사이의 실시간 영향 관계 (하나를 바꾸면 다른 것도 조정 필요)

L3-CA 카메라 ←──────────────────────────────────────┐
  │                                                  │
  ├──> L3-LT 조명: 카메라 위치가 바뀌면 조명 재설정     │
  ├──> L3-CO 구도: 카메라가 구도를 결정                 │
  ├──> L3-PF 퍼포먼스: CU면 표정 중심, WS면 바디 중심   │
  └──> L3-CG 그레이딩: 카메라 렌즈가 컬러 렌디션에 영향   │
                                                      │
L3-LT 조명 ──────────────────────────────────────────│
  │                                                  │
  ├──> L3-CG 그레이딩: 조명이 컬러 베이스를 결정        │
  ├──> L3-AT 대기: 안개/먼지는 빛에 의해 보임           │
  └──> L3-PF 퍼포먼스: 조명이 표정 가시성 결정          │
                                                      │
L3-MU 음악 ──> L3-TR 편집: 음악 비트가 컷 타이밍 결정   │
          ──> L3-SX SFX: 음악과 SFX가 경쟁/보완        │
          ──> L3-PF 퍼포먼스: 음악이 연기 에너지에 영향   │
                                                      │
L3-TR 편집 ──> L3-CN 연속성: 컷 순서가 연속성 요구 결정 ─┘
          ──> L3-CA 카메라: 컷이 바뀌면 다음 샷 앵글 결정
```

## PART 4: 암묵적 L2-L3 경계 결정

| 암묵 결정 | 설명 | 영향 |
|-----------|------|------|
| **커버리지 패턴** | 씬당 마스터+오버숄더+인서트 같은 기본 커버리지 규칙 | L3-CA |
| **키라이트 동기 규칙** | "빛은 항상 물리적 근거가 있어야" vs "감정 우선" | L3-LT |
| **컷 빈도 기준선** | 평균 샷 길이 (액션=2초, 드라마=6초, 예술=15초+) | L3-TR |
| **퍼포먼스 에너지 레인지** | 이 프로젝트의 연기 스펙트럼 (리얼리스틱 vs 과장) | L3-PF |
| **디제틱 사운드 밀도** | 얼마나 많은 환경 소리를 넣는지 (미니멀 vs 맥시멀) | L3-SX |
| **그레이딩 자유도** | 글로벌 팔레트에서 얼마나 벗어나도 되는지 | L3-CG |
| **연속성 엄격도** | 스타일에 따라 다름 (리얼=엄격, 카툰=느슨) | L3-CN |
| **음악-편집 동기화 정도** | 모든 컷이 비트에 맞는지 vs 느슨한 연결 | L3-MU, L3-TR |
| **대기 효과 리얼리즘** | 사실적 안개 vs 스타일화된 컬러 안개 | L3-AT |
| **180도 룰 준수도** | 클래식=절대 / 모던=의도적 위반 허용 | L3-CA, L3-CN |
