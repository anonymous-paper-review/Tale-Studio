# L1 → L2 → L3 전체 도식화

> 작성일: 2026-04-15
> 목적: 변경 비용 기반 4계층 구조의 관계성, 제약 흐름, 교차 의존성을 시각화

## 1. 전체 계층 구조 한 눈에

```
 변경 비용
 (높음)
   ^
   │  ┌─────────────────────────────────────────────────────────────┐
   │  │ L0: 불변 (IMMUTABLE)                                        │
   │  │                                                             │
   │  │  [M] 매체   [R] 해상도   [F] fps   [A] 비율   [E] 렌더     │
   │  └──────────────────────────┬──────────────────────────────────┘
   │                             │ 제약
   │  ┌──────────────────────────▼──────────────────────────────────┐
   │  │ L1: 초고비용 (NEAR-TOTAL REMAKE)                             │
   │  │                                                             │
   │  │  [S] 스타일  [H] 쉐이프  [L] 라인  [C] 캐릭터  [T] 텍스처   │
   │  └──────────────────────────┬──────────────────────────────────┘
   │                             │ 제약
   │  ┌──────────────────────────▼──────────────────────────────────┐
   │  │ L2: 고비용 (PARTIAL MODIFICATION)                            │
   │  │                                                             │
   │  │  [CP] 팔레트  [CM] 컬러의미  [LO] 로케이션  [WD] 의상       │
   │  │  [SD] 사운드  [VX] VFX     [TY] 타이포     [PD] 소품       │
   │  └──────────────────────────┬──────────────────────────────────┘
   │                             │ 제약
   │  ┌──────────────────────────▼──────────────────────────────────┐
   │  │ L3: 유연 (PER-SHOT FLEXIBLE)                                 │
   │  │                                                             │
   │  │  [CA] 카메라  [LT] 조명  [CG] 그레이딩  [PF] 퍼포먼스      │
   │  │  [DL] 대사    [SX] SFX  [MU] 음악큐    [DR] 드레싱         │
   │  │  [AT] 대기    [TR] 편집  [CO] 구도      [CN] 연속성         │
   v  └─────────────────────────────────────────────────────────────┘
 (낮음)
```

## 2. L1 → L2 제약 흐름도

```
L1 ══제약══> L2

[S] 아트 스타일 ═══════════════════════════════════════════════════
     │
     ├══> [CP] 팔레트: 포토리얼=자연 채도 / 셀셰이드=플랫 / 애니메=하이채도
     ├══> [CM] 컬러의미: 포토리얼=미묘 / 카툰=노골적 / 추상=형태 대체
     ├══> [SD] 사운드: 포토리얼=리얼사운드 / 카툰=과장 / 애니메=관례SE
     ├══> [VX] VFX: 포토리얼=투명VFX / 스타일=가시적VFX / 2D=핸드드로운
     └══> [TY] 타이포: 핸드드로운=유기서체 / SF=기하서체 / 러프=디스트레스

[H] 쉐이프 언어 ═══════════════════════════════════════════════════
     │
     ├══> [LO] 로케이션: 원형=둥근건축/아치 / 각진=직선건축/첨탑
     ├══> [WD] 의상: 원형=부드러운옷/흐르는천 / 각진=갑옷/구조적
     └══> [PD] 소품: 쉐이프 규칙 전체 적용 (원형소품/각진소품)

[T] 텍스처 철학 ═══════════════════════════════════════════════════
     │
     ├══> [CP] 팔레트: 그런지=채도 떨어짐 / 프리스틴=깨끗 / 페인터리=블리드
     ├══> [LO] 로케이션: 프리스틴=광택 / 리브드인=얼룩 / 그런지=녹/이끼
     └══> [VX] VFX: 그레인=그레인매칭 / 클린=클린VFX

[C] 캐릭터 코어 ═══════════════════════════════════════════════════
     │
     ├══> [WD] 의상: 프로포션→핏 / 실루엣 유지+강화
     └══> [PD] 소품: 캐릭터 연관 소품이 디자인 언어 따름

[L] 라인 퀄리티 ═══════════════════════════════════════════════════
     │
     ├══> [WD] 의상: 두꺼운선=디테일 단순화 / 라인리스=색/명도 분리
     └══> [TY] 타이포: 러프선=디스트레스타입 / 클린선=클린타입
```

## 3. L2 → L3 제약 흐름도

```
L2 ══제약══> L3

[CP] 컬러 팔레트 ══════════════════════════════════════════════════
     │
     ├══> [CG] 그레이딩: 글로벌 팔레트 = 그레이딩의 "울타리"
     ├══> [LT] 조명: 따뜻한 팔레트 → 따뜻한 조명 경향
     └══> [AT] 대기: 팔레트 내 대기 색상

[CM] 컬러 의미 ════════════════════════════════════════════════════
     │
     ├══> [CG] 그레이딩: 슬픔=블루시프트, 분노=레드시프트 등
     └══> [LT] 조명: 감정 컬러 → 조명 젤/색온도

[LO] 로케이션 ═════════════════════════════════════════════════════
     │
     ├══> [CA] 카메라: 공간 평면도 → 카메라 배치 가능 위치
     │                 천장 높이 → 크레인/하이앵글 가능 여부
     ├══> [LT] 조명: 실용적 광원 → 키라이트 동기/방향
     │                재질 → 반사/흡수
     ├══> [SX] SFX: 재질 → 잔향/폴리 특성
     ├══> [DR] 드레싱: 건축양식 → 어떤 오브젝트가 속하는지
     └══> [AT] 대기: 실내/실외, 환기, 물 근접성

[WD] 의상 ═════════════════════════════════════════════════════════
     │
     ├══> [LT] 조명: 어두운의상=림라이트필요 / 반짝이는의상=스페큘러통제
     ├══> [CA] 카메라: 화려한의상 → FS/MFS로 보여주기
     └══> [PF] 퍼포먼스: 의상이 움직임 제약 (갑옷 vs 드레스)

[SD] 사운드 기조 ══════════════════════════════════════════════════
     │
     ├══> [SX] SFX: 리얼=리얼폴리 / 스타일=과장SE / 미니멀=정적활용
     ├══> [MU] 음악: 장르/스타일 프레임
     └══> [DL] 대사: 리버브 철학 (드라이/공간/과장)

[VX] VFX 스타일 ═══════════════════════════════════════════════════
     │
     ├══> [LT] 조명: VFX 요소 자체가 광원 (폭발, 마법, 홀로그램)
     ├══> [AT] 대기: VFX 파티클 = 대기 효과와 통합
     └══> [CG] 그레이딩: VFX 색상이 글로벌 그레이딩에 반응
```

## 4. L2 내부 교차 의존성 매트릭스

```
           CP    CM    LO    WD    SD    VX    TY    PD
CP 팔레트  --    >>>   >>>   >>>   ~     >>>   >>>   >>>
CM 컬러의미 <<<   --    >>    >>>   ~     >>    >     >>
LO 로케이션 <<<   <<    --    <>    >>    <>    >     >>>
WD 의상     <<<   <<<   <>    --    >     <>    ~     >>
SD 사운드   ~     ~     <<    <     --    >>    ~     <
VX VFX      <<<   <<    <>    <>    <<    --    >     <>
TY 타이포   <<<   <     <     ~     ~     <     --    <
PD 소품     <<<   <<    <<<   <<    <     <>    <     --

>>> = 강한 제약 (팔레트가 의상 색을 지배)
>>  = 중간 제약
>   = 약한 제약
<<< = 역방향 강한 영향
~   = 무시할 수 있는 관계
<>  = 양방향 영향
```

**최고 커플링 쌍 (에이전트 리서치 확인):**
1. **팔레트 <-> 의상**: 의상이 팔레트의 가장 가시적 전달체
2. **팔레트 <-> 로케이션**: 환경이 가장 넓은 컬러 표면
3. **컬러의미 <-> 의상**: 캐릭터 아크의 주요 시각적 차량
4. **로케이션 <-> 사운드**: 공간이 음향을 결정
5. **로케이션 <-> 소품**: 소품이 공간에 속해야 함

## 5. L3 내부 교차 의존성 (샷 단위 실시간 상호작용)

```
L3 요소들의 상호 영향 -- 하나를 바꾸면 다른 것도 조정 필요

[CA] 카메라 ◄─────────────────────────────────────────────────┐
  │                                                           │
  ├──> [LT] 조명: 앵글 바뀌면 조명 재설정 (역광/순광 결정)       │
  ├──> [CO] 구도: 카메라 = 구도 그 자체                         │
  ├──> [PF] 퍼포먼스: CU=표정중심, WS=바디중심                  │
  ├──> [CG] 그레이딩: 렌즈 특성이 컬러 렌디션에 영향             │
  └──> [CN] 연속성: 카메라 무브먼트가 컷 간 매칭 필요            │
                                                               │
[LT] 조명 ────────────────────────────────────────────────────│
  │                                                           │
  ├──> [CG] 그레이딩: 조명 색온도 = 그레이딩 출발점             │
  ├──> [AT] 대기: 안개/먼지 = 빛에 의해서만 가시적 (볼류메트릭)  │
  ├──> [PF] 퍼포먼스: 조명이 표정 가시성 결정                    │
  └──> [CA] 카메라: 저광 → 넓은 조리개 → 얕은 DOF              │
                                                               │
[MU] 음악 ────> [TR] 편집: 비트가 컷 타이밍 결정                │
          ────> [SX] SFX: 음악과 SFX 주파수 경쟁/보완            │
          ────> [PF] 퍼포먼스: 음악 에너지가 연기 에너지에 영향    │
                                                               │
[TR] 편집 ────> [CN] 연속성: 컷 순서 = 연속성 요구 결정 ────────┘
          ────> [CA] 카메라: 다음 컷의 앵글 결정 (30도 룰 등)
          ────> [MU] 음악: 컷 리듬이 음악 리듬과 동기화

[AT] 대기 ────> [LT] 조명: 비 = 반사면 2배 (빛 이중화)
          ────> [PF] 퍼포먼스: 날씨 = 찡그림, 움츠림, 젖은 머리
          ────> [SX] SFX: 비/바람 = 소리 소스
          ────> [CG] 그레이딩: 안개 = 컬러 확산

[PF] 퍼포먼스 ────> [DL] 대사: 표정이 대사 딜리버리와 동기화
              ────> [CA] 카메라: 움직임이 카메라 추적 동기
```

## 7. L0→L1→L2→L3 전체 제약 흐름 (하나의 예시로 추적)

```
"스타일라이즈드 3D 애니메이션으로 느와르 미스터리를 만든다" 라는 프로젝트:

L0 확정:
  [M] 3D CGI (스타일라이즈드)
  [R] 4K, DCI-P3, HDR10
  [F] 24fps, on 2s (액션 on 1s)
  [A] 2.39:1 아나모픽
  [E] 스타일라이즈드 PBR + NPR 합성 (아케인 방식)

     ↓ L0가 L1을 제약

L1 확정:
  [S] 아트 스타일: 하이튼드 리얼리즘 + 페인터리 텍스처
      ← [M] 3D이므로 기본 "CG룩", 의도적 NPR 필요
      ← [E] 스타일라이즈드 PBR이므로 물리 빛 + 페인터리 표면
  [H] 쉐이프: 각진 중심 (느와르 = 삼각형/위협)
      ← [A] 2.39:1 와이드 → 수평 구도 유리
  [L] 라인: 가변 웨이트, 텍스처드 (에지디텍션 + 핸드오버레이)
      ← [M] 3D → 알고리즘 라인 필수
  [C] 캐릭터: 7.5:1 리얼 프로포션, 날카로운 실루엣
      ← [H] 각진 쉐이프 → 각진 얼굴 특징
  [T] 텍스처: 리브드인/웨더드, 핸드페인트, GGX BRDF
      ← [S] 페인터리 → 붓터치 가시적

     ↓ L1이 L2를 제약

L2 설정:
  [CP] 팔레트: 틸/앰버 2색 기조, 뮤트 채도, 로우키
      ← [S] 하이튼드 리얼 → 자연 범위 내 but 스타일화 허용
      ← 느와르 장르 → 하이콘트라스트, 어두운 톤
  [CM] 의미: 앰버=위험/욕망, 틸=고독/진실, 빨강=폭력(금지색 수준으로 제한)
  [LO] 로케이션: 아르데코 + 브루탈리즘, 좁은 통로, 높은 천장
      ← [H] 각진 → 직선 건축
      ← [T] 리브드인 → 벽에 얼룩, 긁힘
  [WD] 의상: 트렌치코트, 구조적 수트, 어두운 톤
      ← [H] 각진 → 날카로운 칼라
      ← [CP] 뮤트 팔레트 → 채도 낮은 의상
  [SD] 사운드: 리얼리스틱 + 스타일화된 앰비언스 (빗소리 과장)
      ← [S] 하이튼드 리얼 → 리얼 기반이지만 과장 허용
  [VX] VFX: 미니멀, 안개/비/네온 반사 중심
      ← [S] 느와르 → 보이지 않는 VFX, 대기 효과 중심

     ↓ L2가 L3를 제약

L3 (각 샷에서):
  [CA] 카메라: 느린 돌리, 로우앵글, 85mm 압축, 얕은 DOF
      ← [LO] 좁은 통로 → 카메라 위치 제한
      ← [A] 2.39:1 → 극단 수평 프레이밍
  [LT] 조명: 4:1~8:1 비율, 사이드라이팅, 네온 프랙티컬
      ← [LO] 네온 간판/가로등 → 광원 동기
      ← [CP] 틸/앰버 → 조명 색온도 결정
  [CG] 그레이딩: 디새추레이트, 틸앤앰버, 크러시드 블랙
      ← [CP] 팔레트 울타리 내에서
      ← [CM] 고독 씬 = 틸 강화, 위험 씬 = 앰버 강화
  [PF] 퍼포먼스: 억제된 표정, 턱 긴장, 느린 시선 이동
      ← 느와르 장르 규칙 + 스릴러 레시피
  [TR] 편집: 느린 패이싱 + 교차 편집 + J컷
      ← 스릴러 레시피
  [AT] 대기: 비, 안개, 네온 반사, 젖은 거리
      ← [VX] 대기 효과 중심
      ← [LO] 도시 로케이션
```
