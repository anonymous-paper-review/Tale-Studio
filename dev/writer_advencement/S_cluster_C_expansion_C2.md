## 2.7 마이크로 Hook

### 심리학적 토대
- **자이가르닉 효과** (1927): 미완료 과제는 완료된 것보다 뇌에서 더 강하게 활성화
- **정보 갭 이론** (Loewenstein, 1994): 아는 것과 알고 싶은 것의 간극 = "인지적 가려움"
- **인지 부조화** (Festinger, 1957): 기대와 현실의 불일치 = 해소를 위해 계속 주의

**핵심**: 매 순간 최소 1개의 open loop. 하나를 닫으면 다른 것을 열어야.

### 10가지 유형

| # | 유형 | 메커니즘 | 클리셰 경계 |
|---|------|---------|-----------|
| 1 | **호기심 갭** | 부분 정보 → 완결 요구 | 미스터리 박스 남용 (답 없이 질문만 쌓으면 좌절) |
| 2 | **미완료 행동** | 행동 시작 → 결과 전 컷 | "매번 1초 전 해제" = 3회차에 예측 가능 |
| 3 | **끊긴 대사** | 핵심 정보 직전 중단 | "나중에 말할게"로 인위적 보류 → 캐릭터가 바보 |
| 4 | **설명 없는 디테일** | 맥락 없는 이상한 요소 | 1/3 이상은 페이오프 필요, 아니면 잡음 |
| 5 | **미세 불일치** | 기대와 다른 반응 | "쿨한 사이코" 남용. 내면 논리와 일관해야 |
| 6 | **시각적 미끼** | 프레임 주변부에 주의 끄는 요소 | TV 뉴스 복선 = 극도 과용. 미묘함이 핵심 |
| 7 | **시간 압박** | 카운트다운, 마감, 긴급 | **가장 남용.** 모호한 긴급성 > 정확한 초 |
| 8 | **약속/예고** | "이건 시작에 불과해" | 과대 약속 → 실망 페이오프. BB 시즌2 곰인형=효과적 |
| 9 | **패턴 파괴** | 확립된 패턴을 갑자기 깨뜨림 | 최소 3회 반복 후 깨야 인지. 단순 점프스케어 ≠ |
| 10 | **감각적 끌림** | 논리 무관, 시청각 매력으로 포획 | "스타일>서브스턴스" 과잉 주의 |

### 운용 규칙

```
1. 동시 open loop: 최소 1, 최대 3~4. (0=이탈, 5+=과부하)
2. Loop 수명: 마이크로(수초) / 씬(수분) / 매크로(전체) 중첩
3. 닫힘↔열림 교대: 하나 닫을 때 다른 것 열기
4. Hook 다양성: 같은 유형 3회 연속 금지 (인지↔감각↔감정 순환)
5. Earned Hook: 페이오프 없는 Hook = 신뢰 훼손
```

### 밀도 가이드

| 콘텐츠 | 매 N초마다 | 동시 loop | 주요 유형 |
|--------|----------|----------|---------|
| 숏폼 (15~60초) | 3~5초 | 1~2 | 호기심갭, 시각미끼, 시간압박 |
| 미드폼 (2~5분) | 10~20초 | 2~3 | 전 유형 순환 |
| 롱폼 (10분+) | 20~40초 | 2~4 | 미완료행동, 패턴파괴, 약속 |

## 2.8 씬 브레이크 / 클리프행어

### 6가지 유형

| 유형 | 메커니즘 | 장르 친화도 |
|------|---------|-----------|
| **미해결 질문** | 강한 질문 제기, 답 없이 씬 종료 | 미스터리, 스릴러, 드라마 |
| **행동 중단** | 추격/전투 한가운데에서 컷 | 액션, 스릴러, 호러 |
| **반전 브레이크** | 마지막 순간에 모든 것 재맥락화 | 스릴러, 미스터리, SF |
| **감정적 절벽** | 극한 감정 상태에서 해소 없이 종료 | 드라마, 로맨스 |
| **정보 폭탄** | 파괴적 정보 전달 직후, **반응 미공개** | 전 장르 |
| **거짓 해결** | 해결된 듯 → 한 디테일이 전부 무너뜨림 | 스릴러, 호러 |

### 운용 규칙

```
1. 전체 씬의 40~60%만 강한 Hook으로 끝낼 것 (나머지는 soft landing)
2. 같은 유형 연속 금지 (질문형→행동형→감정형→반전형 순환)
3. 초반 < 중반 < 클라이맥스 전 강도 에스컬레이션
4. 클리프행어로 연 질문은 반드시 가까운 미래에 해소 시작
```

## 2.9 극적 질문 (Dramatic Question)

### CDQ (Central Dramatic Question)

전체 이야기가 존재하는 이유인 하나의 질문. inciting incident에서 제기, climax에서 답.

**5가지 속성**: 명확성, 개인적 stakes, 불확실성, 보편성, 긴급성.
5개 중 하나라도 약하면 관객이 "그래서 요점이 뭔데?" 반응.

### 질문의 위계

```
CDQ: "프로도가 반지를 파괴할 것인가?"
  └── Act Q: "원정대가 모리아를 통과할 수 있을까?"
       └── Scene Q: "간달프가 발록을 막을 수 있을까?"
            └── Micro Q: "다리가 버틸까?"
```

**규칙**: 모든 하위 질문은 상위에 봉사. CDQ 무관 씬 질문 = 탈선.

### 장르별 CDQ 패턴

| 장르 | CDQ 구조 |
|------|---------|
| 액션 | "주인공이 X를 달성할 것인가?" |
| 로맨스 | "두 사람이 함께할 것인가?" |
| 미스터리 | "진실/범인이 무엇/누구인가?" |
| 스릴러 | "주인공이 생존할 것인가?" |
| 호러 | "누가 살아남는가?" + "정체는?" |
| 비극 | "주인공이 결함을 극복할 것인가?" (보통 No) |

## 2.10 클리셰 분석

### Hook → 클리셰 진화 단계

```
발명 → 모방 → 관습 → 피로 → 클리셰 → 패러디/메타
(Psycho)  (슬래셔)  (장르기대)  ("또?")   (게으른표지)  (Scream)
```

### Convention vs Cliché 판별

| | Convention | Cliché |
|---|-----------|--------|
| 관객 반응 | "이런 게 있어야지" | "또 이거야" |
| 예측 가능성 | 구조 예측 가능, 실행 신선 | 구조도 실행도 예측 |
| 제거 시 | 이야기 약해짐 | 이야기 그대로 |

### 대표 클리셰 (장르별)

| 장르 | 클리셰 | 대안 |
|------|--------|------|
| 액션 | 폭발 뒤 걸어나오기, 1초전 해제 | 결과 있는 폭발, 부정확한 시간 |
| 호러 | 점프스케어=공포, "혼자 가보겠어" | 슬로우번, 합리적 캐릭터 |
| 로맨스 | 공항 달리기, 비속 키스 | 일상적 친밀함 |
| 미스터리 | 탐정이 모아놓고 설명 | 관객과 동시 발견 |

### 클리셰 자가 테스트 5개

1. **첫 아이디어 테스트**: 가장 먼저 떠오른 것 = 대부분 클리셰. 3개 대안 중 가장 예상 못한 것 선택.
2. **검색 테스트**: TV Tropes에서 검색 가능한가? → 변형 필요.
3. **캐릭터 특수성**: 다른 캐릭터에 넣어도 성립 = generic = 클리셰 방향.
4. **제거 테스트**: 빼도 이야기 그대로 = 장식 클리셰.
5. **관객 예측**: 경험 많은 관객이 다음을 맞출 수 있으면 변형 필요.

### 전복 전략 5가지

| 전략 | 사례 |
|------|------|
| 지연 전복 | No Country: 영웅이 빌런과 대면 안 함 |
| 결과 전복 | Casino Royale: 본드가 실제로 고통받음 |
| 시점 전복 | Rosencrantz & Guildenstern: 단역 시점 |
| 메타 전복 | Scream: "호러에서 살려면 규칙이 있어" |
| 진정성 전복 | Manchester by the Sea: 영웅적 극복 대신 현실적 무너짐 |

**전복의 3전제**: 장르 지식 + 관객 기대 구축 + 의미 (gotcha가 아닌 깊은 진실)
