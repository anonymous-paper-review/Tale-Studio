## 1. 템포 조절 (Tempo Control)

### 1.1 씬 길이와 서사 기능

| 씬 유형 | 지속 시간 | 이유 |
|---------|---------|------|
| 엑스포지션 | 김 | 세계/캐릭터 소개에 시간 필요. 관객을 이야기로 초대 |
| 액션 | 짧음 | 빠른 컷 = 갈등/에너지. 짧은 샷이 긴장 생성 |
| 감정 여파 | 중간~김 | 결정/클라이맥스 후 관객에게 소화 시간 제공 |
| 대화 | 중간 | 대사 밀도에 따라 가변 |

**영화 전체 템포 아크**:
느리게 시작(엑스포지션) → 가속(일반 서사 진행) → 살짝 감속(서사 복잡해짐) → 클라이맥스 질주 → 감속(일상 복귀)

### 1.2 시간 압축과 확장

| 기법 | 효과 | 사용 시점 |
|------|------|----------|
| **몽타주** | 며칠/몇주를 초 단위로 압축 | 훈련, 시간 경과, 병렬 발전 |
| **엘립시스** | 불필요한 행동 생략 | 로케이션 전환, 루틴 건너뛰기 |
| **타임 스킵** | 큰 시간 도약 | 노화, 시간에 걸친 결과 |
| **슬로 모션** | 짧은 순간을 늘림 | 감정 피크, 충격, 주관적 시점 |
| **리얼타임** | 1:1 비율 | 긴박함, 밀실, 즉시성 (버드맨) |
| **늘어난 순간** | 짧은 사건을 다중 앵글로 | 사이코 샤워 씬: 30초 스토리 → 1분+ 스크린 |

슬로 모션은 객관→주관 시점 전환. 캐릭터의 눈으로 세계를 경험하게 하여 공감 증대.

### 1.3 대사 밀도

**소킨 모델 (고밀도)**: 대사를 음악 작곡처럼 처리. 케이던스(리듬) + 톤(감정 키) + 볼륨(다이내믹). 배우가 음절 하나를 빠뜨리면 "음악"이 깨짐.

**명상적 모델 (저밀도)**: 말릭, 오즈. 시각적 스토리텔링과 침묵이 감정 전달. 의미 있는 정지와 톤 변화로 관심 유지.

| 도구 | 효과 |
|------|------|
| 빠른 대화 교환 | 긴박, 지성, 갈등 |
| 숙고하는 전달 | 무게, 권위, 슬픔 |
| 정지와 침묵 | 감정 무게, 처리, 파워 다이내믹 |
| 끼어듦/인터럽트 | 갈등, 무례, 흥분, 혼란 |
| 겹치는 대사 | 자연주의, 그룹 다이내믹, 에스컬레이팅 긴장 |

### 1.4 의도적 감속

**미야자키의 마(間)**:
- 액션 사이의 공간, 결정을 생각할 여유.
- "논스톱 액션만 있으면 그냥 분주함일 뿐. 잠깐 멈추면 긴장이 더 넓은 차원으로 성장."
- "80도의 일정한 긴장은 무감각만 만든다."
- 관객이 반성하고, 상상하고, 해석하게 함. 대사로 감정을 말하는 대신 관객이 캐릭터의 감정을 함께 경험.

**빌드업 원리**: 느린 것이 빠른 것을 더 빠르게 느끼게 함. 끊기지 않는 긴 테이크가 관객에게 실시간 흡수를 허용하고, 가속이 오면 대비가 충격을 증폭.

**감속이 작동할 때 vs 모멘텀을 죽일 때**:
- 작동: 분위기 구축, 감정 심화, 공포/기대 생성, 캐릭터 내면성
- 실패: 관객이 투자할 이유가 없을 때, 아무것도 전달되지 않을 때, 의도가 아닌 실수일 때

### 1.5 의도적 가속

- **몽타주**: 핵심 순간의 빠른 연속으로 시간 압축 + 감정 구축
- **교차 편집**: 다른 장소의 동시 액션 = 긴장. 병렬 편집이 비교를 통해 극적 아이러니 생성
- **클라이맥스 접근 시 컷 빈도 증가**: 긴 샷→점진적 단축. 사이코: 공격 전 느린 편집 → 커튼 열리면 급가속

**속도가 지치게 할 때**: 변화 없는 빠른 컷 = 무감각. 대비 원칙: 속도도 간헐적 쉼이 있어야 충격 유지.

### 1.6 장르별 템포 패턴

| 장르 | 패턴 | 원리 |
|------|------|------|
| 호러 | 느림-느림-느림-**폭발** | 정적과 고요가 충격을 효과적으로 |
| 코미디 | 셋업-셋업-**펀치라인**-비트 | 리듬과 서프라이즈; 웃음 뒤 "비트" 필수 |
| 스릴러 | 꾸준한 가속 + 주기적 정체 | 지속 긴장 + 짧은 회복 = 피로 방지 |
| 드라마 | 밀물과 썰물, 전반적 느림 | 머무름이 감정 공명 생성 |
| 액션 | 빠른 기준선 + 짧은 회복 | 대비가 빠른 시퀀스를 더 빠르게 |

## 2. 정보 은폐/드러냄 (Information Concealment/Revelation)

### 2.1 미스디렉션

| 유형 | 메커니즘 | 예시 |
|------|---------|------|
| **시각적** | 카메라 포커스가 시선 유도. 셸로우 포커스로 한쪽 집중, 다른 쪽 은폐 | 배경에 흐릿한 인물 접근 |
| **서사적** | 스토리가 결론 A를 암시하지만 진실은 B | "더 스팅": 반전이 선행 사건 이해를 뒤바꿈 |
| **캐릭터 기반** | "신뢰할 만한" 캐릭터가 거짓이거나 무의식적으로 오류 | 동맹으로 보이는 인물이 적대자 |

### 2.2 레드헤링

**페어 플레이 룰**: 모든 레드헤링은 공정해야. 진짜 단서가 레드헤링과 함께 존재해야 함. 정보 은폐로만 작동하는 레드헤링 = 치팅.

**비율 규칙**: 핵심 단서 1개당 설득력 있는 레드헤링 2개 (1:2 비율).

**해소 필수**: 레드헤링은 결국 거짓으로 밝혀져야. 미해소 = 떨어뜨린 플롯 스레드.

**관객이 배신감 느끼는 때**: 레드헤링이 너무 많고 진짜 단서 부족 / 불가능한 추론 요구 / 진짜 감정 투자를 사소화

### 2.3 신뢰불가 화자 (Unreliable Narrator)

| 유형 | 메커니즘 | 사례 |
|------|---------|------|
| **자기보호자 (거짓말쟁이)** | 의도적 기만, 자기 보호 | 유주얼 서스펙트 (버벌 킨트), 나를 찾아줘 (에이미) |
| **광인** | 정신 질환이 지각 왜곡 | 파이트 클럽, 셔터 아일랜드, 블랙 스완 |
| **순진한 자** | 의도 없이 이해 부족 | 포레스트 검프, 아동 화자 |
| **광대** | 아이러니와 모호함으로 진실 가지고 놈 | 롤리타 (험버트 험버트) |
| **과장자** | 사건을 부풀림 | 회고록 화자, 이야기꾼 |

**신뢰불가 신호 심기**: 미묘한 비일관성, 모순, 이상한 디테일을 심어서 재감상 시에야 보이게.

### 2.4 의도적 생략 (Ellipsis)

| 유형 | 설명 | 사례 |
|------|------|------|
| 시간적 | 루틴/불필요한 시간 건너뛰기 | 의자에서 일어남 → 컷 → 이미 문 앞 |
| 극적 | 중요 사건 생략, 여파로 드러냄 | 오즈: 핵심 사건 생략, 이후 대화로 밝힘 |
| 대규모 | 거대한 시간 도약 | 2001: 뼈 무기 → 우주선 |

**예술적 생략 vs 혼란스러운 생략**:
- 예술적: 관객이 추론 가능 (또는 생산적으로 불확실). 갭이 이해를 풍부하게.
- 혼란: 인과 연결 빠짐. 갭이 좌절을 생성.

### 2.5 은폐의 층위

| 층위 | 설명 | 관객 효과 |
|------|------|----------|
| 캐릭터 → 캐릭터 | 작중 비밀 (A가 아는 것을 B가 모름) | 관객이 비밀을 알 수도 모를 수도 |
| 서사 → 관객 | 스토리텔링 선택 (관객에게 숨김) | 미스터리, 서프라이즈, 반전 |
| 캐릭터 → 자신 | 자기기만, 심리적 부정 | 관객이 캐릭터의 자기기만을 보면 극적 아이러니 |

**층위가 상호작용할 때 가장 강력**: 관객은 A가 B에게 숨기는 비밀을 알고 + A가 자신에게도 숨기고 있음을 봄 → 다중 아이러니 레이어.

### 2.6 드러냄 타이밍 설계

| 타이밍 | 효과 | 리스크 |
|--------|------|--------|
| **초기 드러냄** | 관객이 전체 정보로 시청 = 지속 서스펜스 (히치콕 모델) | 미스터리 소멸 |
| **미드포인트 드러냄** | 이전 모든 것을 재맥락화 = 내러티브 피벗 | 전반부 셋업이 충분해야 |
| **후반 드러냄** | 최대 서프라이즈 = 전체 인식 반전 | 단서 없었으면 치팅 느낌 |
| **점진적 드러냄** | 조각을 시간에 걸쳐 = 미스터리와 몰입 지속 | 너무 느리면 흥미 상실 |

**빵부스러기 원칙**: 충격적 반전이 있는 이야기도, 재감상 시 첫 감상에서 놓친 단서를 보는 것이 쾌감. 은폐와 드러냄이 스토리에 본질적이어야 -- 임의적 서프라이즈가 아니라.

## 3. 반복과 변주 (Repetition and Variation)

| 기법 | 메커니즘 | 효과 |
|------|---------|------|
| **리프레인** | 같은 요소 반복 | 의례감, 필연성, 영속성 |
| **반복 속 변주** | 반복할 때마다 조금씩 변화 | 아크의 시각화, 점진적 변화 전달 |
| **구조적 미러링** | 씬 A와 씬 A'가 대응 (북엔딩) | 변환 보여줌, 내러티브 응집 |
| **3의 법칙** | 첫째 확립, 둘째 강화, 셋째 완성 or 파괴 | 최소한의 패턴 → 만족 or 전복 |
| **에스컬레이팅 반복** | 같은 행동, 위험 증가 | 긴장 축적 |
| **콜 앤 리스폰스** | 한 캐릭터 행동이 다른 캐릭터에 의해 반향/응답 | 대화, 병렬 플롯, 주제적 논증 |

**키아스믹 구조**: A-B-C-B'-A' 패턴으로 중간점이 대칭축. 시작과 끝이 거울이 되는 것을 넘어 전체 구조가 대칭.

## 4. Show vs Tell

**핵심**: "보여주기"가 거의 항상 감정과 캐릭터에 더 강력. 관객이 직접 해석하고 경험.

| 보여주기 기법 | 어떻게 |
|-------------|--------|
| 미장센 | 프레임 안 모든 것: 소품, 세트, 의상, 블로킹 |
| 촬영 기법 | 구도, 블로킹, 조명, 편집이 시각적으로 드러냄 |
| 상징적 요소 | 소품, 반복 색상, 카메라 앵글이 말 없이 전달 |
| 행동과 반응 | 슬픔을 설명하는 대신: 광활한 빈 방, 어두운 빛, 구부러진 어깨 |

**Tell이 정당화되는 때**:
- 복잡한 세계관 규칙 (효율적으로 보여줄 수 없는 정보)
- 시간 압축 (나레이션이 수년을 한 문장으로)
- 전략적 백스토리 (적시에 최대 충격)

**"As You Know, Bob" 문제**: 둘 다 아는 사실을 대화로 설명 → 관객이 가짜 느낌.
해결: 정보를 얇게 퍼뜨리기 / 대사 밖으로 빼기 / 모르는 캐릭터 활용 / 서브텍스트 활용 / 그 정보가 정말 필요한지 재고.

## 5. 클리셰 분석

### 템포 클리셰

| 클리셰 | 왜 실패 | 새롭게 하는 법 |
|--------|---------|-------------|
| 클라이맥스에서 무조건 슬로모 | 과용으로 "여기서 감정 느끼세요" 신호가 됨 | 예상 못한 순간에 드물게, 예상 못한 오디오와 조합 |
| 빠른 컷 = 흥분 | 관객 피로, 편집 리듬이 소음 | 리듬 변화; 고요로 속도를 충격적으로 |
| 업비트 음악 + 훈련 몽타주 | 공식화되어 패러디로 인식 | 몽타주 형식 안에서 기대 전복 |

### 정보 은폐 클리셰

| 클리셰 | 왜 실패 | 새롭게 하는 법 |
|--------|---------|-------------|
| "전부 꿈이었다" | 관객 배신감 -- 투자 시간이 무의미 | 인셉션: 꿈이 반전이 아니라 전제. 현실 불확실 자체가 주제 |
| "범인은 착한 캐릭터" | 너무 예상되어 반전 아님 | 드러냄이 캐릭터 발전에 봉사하게 |
| "악당이 계획을 설명" | 순수 엑스포지션 덤프; 캐릭터 논리 위반 | 영웅의 발견으로 계획 드러냄, 또는 계획이 실행되는 모습을 보여줌 |
| "선택받은 자" | 예측 가능한 운명이 긴장 제거 | 비용, 모호함, 또는 "선택받음" 역할 거부 추가 |

### 현대 관객 인식

현대 관객은 많이 읽고 많이 봐서 흔한 시나리오를 수없이 경험. 과용된 트로프 = 예측 가능성 = 몰입 사망. 관객은 기대가 충족되되 **놀라운 방식으로** 충족되길 원함.

### 익숙한 기법을 신선하게 쓰는 법

1. 트로프를 인정한 뒤 전복
2. 목적을 겹치기 (레드헤링이 캐릭터도 발전시키기)
3. 장르 관습을 깊이 알아야 지적으로 가지고 놀 수 있음
4. 기법 조합 (신뢰불가 화자 + 구조적 미러링 + 에스컬레이팅 반복)
5. 플롯 기교보다 캐릭터 진실 우선
