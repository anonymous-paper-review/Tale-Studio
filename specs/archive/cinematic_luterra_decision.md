# Cinematic Plan: 루테란의 결정

> 카제로스 봉인 직전, 에스더들과의 논쟁

## 개요

| 항목 | 내용 |
|------|------|
| **제목** | 결정의 순간 (The Decisive Moment) |
| **시대** | 사슬전쟁 말기 (현재로부터 500년 전) |
| **장소** | 전쟁터 폐허 / 에스더 회의장 |
| **분위기** | 무거움, 긴장, 갈등, 결단 |
| **예상 길이** | 60-90초 (7-10개 씬) |

## 씬 구성

### Act 1: 상황 설정 (15-20초)

#### Scene 1: 전쟁터 폐허 (Establishing Shot)
```
Shot Type: Wide Shot → Slow Pan
Duration: 8초
Description:
  - 황폐한 전쟁터, 불타는 잔해
  - 하늘은 붉은 노을과 검은 연기
  - 멀리 악마의 성채가 보임
Mood: 절망적이지만 결전 직전의 고요함
Prompt Keywords: battlefield ruins, burning debris, red sunset, dark smoke, demonic fortress silhouette
```

#### Scene 2: 에스더들의 모임 (Group Shot)
```
Shot Type: Medium Shot
Duration: 6초
Description:
  - 원형으로 모인 에스더들
  - 루테란이 중앙에 서있음
  - 각자 지친 표정, 하지만 결의에 찬
Mood: 긴장된 회의 분위기
```

---

### Act 2: 갈등 (30-40초)

#### Scene 3: 아제나의 분노 (Close-up)
```
Shot Type: Close-up → Medium Shot
Duration: 8초
Description:
  - 아제나가 주먹을 쥐며 외침
  - "카제로스를 소멸시켜야 합니다!"
  - 분노에 찬 표정, 눈가에 눈물
Mood: 격렬한 감정, 트라우마
Camera: 약간 흔들림 (감정 강조)
```

#### Scene 4: 루테란의 경청 (Reaction Shot)
```
Shot Type: Medium Close-up
Duration: 5초
Description:
  - 아제나의 말을 듣는 루테란
  - 무거운 표정, 고개를 살짝 숙임
  - 패자의 검을 쥔 손에 힘이 들어감
Mood: 무게감, 책임감
```

#### Scene 5: 아제나 vs 루테란 (Two-Shot)
```
Shot Type: Over-the-shoulder / Two-shot
Duration: 10초
Description:
  - 마주보는 두 사람
  - 아제나: "왜 머뭇거리십니까!"
  - 루테란: 조용히 고개를 든다
Mood: 대립, 긴장의 정점
```

#### Scene 6: 카단의 시선 (Cutaway)
```
Shot Type: Medium Shot
Duration: 4초
Description:
  - 옆에서 지켜보는 카단
  - 차분하지만 의미심장한 표정
  - 루테란을 믿는다는 듯한 눈빛
Mood: 신뢰, 관조
```

---

### Act 3: 결정 (20-30초)

#### Scene 7: 루테란의 결심 (Hero Shot)
```
Shot Type: Low Angle → Medium Shot
Duration: 8초
Description:
  - 루테란이 천천히 고개를 든다
  - 패자의 검을 들어올림
  - "소멸시키면... 더 큰 희생이 따릅니다"
Mood: 결단, 영웅적 순간
Lighting: 뒤에서 빛이 비추는 실루엣
```

#### Scene 8: 에스더들의 반응 (Group Reaction)
```
Shot Type: Wide Shot (얼굴들 훑기)
Duration: 6초
Description:
  - 각 에스더들의 반응
  - 아제나: 충격과 분노
  - 카단: 고개를 끄덕임
  - 다른 에스더들: 복잡한 표정
Mood: 혼란, 하지만 따르겠다는 결의
```

#### Scene 9: 루테란의 선언 (Close-up)
```
Shot Type: Extreme Close-up (눈) → Pull back
Duration: 8초
Description:
  - 루테란의 눈에서 결의가 빛남
  - "우리의 역할은 봉인으로 끝납니다"
  - 검을 땅에 꽂으며 결정을 선포
Mood: 역사적 순간, 무거운 결단
```

---

### Act 4: 여운 (10-15초)

#### Scene 10: 엔딩 (Wide Shot)
```
Shot Type: Wide Shot → Fade to Black
Duration: 10초
Description:
  - 에스더들이 루테란 주위로 모임
  - 멀리서 카제로스의 성채가 보임
  - 하늘에 희미한 빛이 비치기 시작
Mood: 결정 후의 무거운 고요, 희망의 씨앗
Music: 장엄한 오케스트라 (피아노 → 풀 오케스트라)
```

## 기술 명세

### 영상 설정
| 항목 | 값 |
|------|-----|
| Aspect Ratio | 16:9 |
| Duration per shot | 5-8초 |
| Total shots | ~10개 |
| Generation Method | T2V (Text-to-Video) |

### API 사용 예상
| 항목 | 수량 |
|------|------|
| Veo 생성 | 10회 |
| 필요 API 키 | 1개 (10/일 한도 내) |
| 예상 소요 시간 | 30-40분 |

### 스타일 가이드
```
Base Style Keywords:
- Lost Ark game cinematic style
- Epic fantasy, dark atmosphere
- Korean MMO aesthetic
- High detail armor and weapons
- Dramatic lighting (rim light, volumetric)
- Cinematic color grading (desaturated with accent colors)
```
