# L1: Scene Architect

> Pumpup (전처리) + Scene 분할

## 역할

스토리 텍스트를 받아 시각화 정보를 보강(Pumpup)한 뒤, 씬 단위로 분할하고 캐릭터/로케이션을 정의한다.

```
Story Input → [Pumpup] → Expanded Story → [Scene Architect] → Scene Manifest
```

**UX 매핑**: P2 The Script Room — AI 작가와 채팅으로 씬 구성/수정 (`specs/ux_pages.md` P2)

---

## Pumpup (전처리)

짧은 스토리 입력에 **L1이 씬 분할하기 좋고, 영상 API가 시각화할 수 있는 정보**를 추가.

**핵심**: 서사 보존 + 시각화 정보 확장 (문학적 확장 X)

### 추가하는 정보

| 카테고리 | 설명 | 예시 |
|----------|------|------|
| 시간/조명 | 시간대, 광원, 빛 방향 | "황혼녘", "석양빛이 비스듬히 내리쬐는" |
| 장소 구체화 | 공간 규모, 배경 요소, 지면 | "시체가 널린 피로 물든 평원" |
| 물리적 동작 | 속도, 방향, 신체 부위 | "천천히 검을 들어올렸다" |
| 환경 디테일 | 공기, 날씨, 분위기 요소 | "바람에 나부끼는 망토" |

### 추가하지 않는 것

| 카테고리 | 이유 |
|----------|------|
| 감정→시각 표현 | 캐릭터성에 따라 다름, 룰 기반 불가 |
| 캐릭터 내면 | "가슴 속 분노" 등 영상 API가 못 그림 |
| 캐릭터성 반영 | 맥락 의존적, L2 또는 상위 레벨에서 처리 |

### 보존 규칙

- **MUST**: 고유명사, 핵심 대사 (원문 그대로), 인과관계, 전개 순서
- **MUST NOT**: 캐릭터 추가, 스토리 전개 변경, 원작 설정 충돌

### 인터페이스 (개념)

> 아래는 입출력 계약 정의. 구현은 Next.js 기반으로 새로 작성 예정.

- **입력**: story (100~500자), lore_context (참조 로어/설정), source_title (외부 레퍼런스 검색용), target_length (목표 1500자)
- **출력**: 시각화 정보가 추가된 스토리 (1000~2500자)
- `source_title` 제공 시 웹검색으로 외부 레퍼런스(게임/애니 위키 등) 자동 조회
- 감정 단어는 변환하지 않고 유지, LLM 판단 로그만 기록

## Scene Architect

### 입력

| 항목 | 소스 |
|------|------|
| Expanded Story | Pumpup 출력 (or 원본 스토리) |
| 캐릭터 정보 | 사용자 입력 (이미지 + 설명) |
| 프로젝트 설정 | 제작 길이, 장르, 화면 비율 |

### 출력

| 출력물 | 내용 |
|--------|------|
| `scene_manifest` | 씬 목록, 유형, 길이, Act 배분 (기승전결) |
| `character_sheet` | 캐릭터 ID, Fixed Prompt, 레퍼런스 이미지 경로 |
| `location_sheet` | 로케이션 정의, 시각 정보 |

### 씬 구조

```
Story (1개)
  └── Scene (4개, 기승전결)
        └── 각 씬: narrative_summary, location, characters, mood, original_text
```

### LLM 프롬프트가 추출하는 것

- `narrative`: 씬 요약
- `location`: 장소
- `original_text`: 원문 인용
- `characters`: 등장 캐릭터
- `mood`: 분위기

## 예시

### Pumpup 입/출력

**입력** (150자):
```
루테란은 전장에서 홀로 살아남았다. 동료들의 시체 사이에 서서 패자의 검을
들어올린다. "이 싸움은 끝나지 않았다."
```

**출력** (500자):
```
황혼이 내려앉은 전장에는 수천 구의 시체가 널려 있었다. 한때 푸르렀던
평원은 이제 핏빛으로 물들어 있었다. 석양빛이 비스듬히 내리쬐어 긴
그림자를 드리웠다.

그 한가운데, 루테란이 서 있었다. 바람이 불어와 그의 망토가 나부꼈다.
발밑에는 함께 싸웠던 동료들이 쓰러져 있었다.

천천히, 루테란은 패자의 검을 들어올렸다. 검날이 석양빛을 받아 금빛으로
빛났다. 검날에 맺힌 핏방울이 땅으로 떨어졌다.

"이 싸움은 끝나지 않았다."
```

## 결정 사항

- Pumpup은 L1 전처리로 채택 (옵션 A)
- 목표 분량 기본값: 1500자 (범위 1500~2000)
- 로어 컨텍스트: source_title로 웹검색 (외부 레퍼런스 있는 경우만)
- 감정 단어: 배제 (변환 안 함, 로그만)
